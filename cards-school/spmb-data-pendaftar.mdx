# SPMB — Data Pendaftar

Menu Data Pendaftar digunakan Admin untuk mengelola seluruh data calon peserta didik yang mendaftar melalui website [SPMB](/referensi/glosarium#spmb).

## List Data Pendaftar

Admin memiliki kemampuan untuk mengelola list data pendaftar SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pada halaman Data Pendaftar, Admin dapat:
   - Melihat daftar data pendaftar SPMB.
   - Import data pendaftar.
   - Export data pendaftar.
   - Search data pendaftar.
   - Filter data pendaftar.

## Edit Pendaftaran

Admin memiliki kemampuan untuk mengedit data pendaftaran:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Klik tanda titik tiga pada baris data pendaftar yang ingin diedit.
5. Klik **Edit**.
6. Sistem akan menampilkan detail data pendaftar.
7. Edit formulir sesuai kebutuhan.
8. Klik **Simpan** untuk menyimpan data yang telah diperbarui.

## Detail

Admin memiliki kemampuan untuk memonitor detail data pendaftar:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Klik pada data pendaftar.
5. Sistem akan menampilkan detail data pendaftar yang dipilih.
6. Admin dapat **Cetak Detail Pendaftar**.
7. Admin dapat **Edit Detail Pendaftar**:
   - Gulir detail pendaftar hingga bawah.
   - Klik opsi **Edit** pada Detail Pendaftar.
   - Isi form sesuai kebutuhan.
   - Klik **Simpan** untuk menyimpan pembaruan.

## Cetak

Admin memiliki kemampuan untuk mencetak detail data pendaftar:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Klik pada data pendaftar yang ingin dicetak.
5. Sistem akan menampilkan detail data pendaftar.
6. Gulir detail pendaftar hingga bawah.
7. Pilih opsi **Cetak** untuk mencetak data detail pendaftar.
8. Sistem akan menampilkan tampilan Print:
   - Klik **Print** untuk mencetak detail data pendaftar langsung melalui printer yang terhubung.
   - Admin dapat mengunduh dan menyimpannya ke perangkat.

## Lihat Invoice

Admin memiliki kemampuan untuk mengelola data pembayaran SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pilih submenu **Aksi** pada nama siswa.
5. Klik **Lihat Invoice**.
6. Sistem akan menampilkan Data Pembayaran.
7. Pada halaman tersebut, Admin dapat:
   - Memonitor ringkasan pembayaran.
   - Memonitor daftar riwayat pembayaran.
   - Cetak bukti pembayaran.
   - Konfirmasi pembayaran tunai.
   - Ubah metode pembayaran.

## Konfirmasi Pembayaran

Admin memiliki kemampuan untuk melakukan konfirmasi pembayaran tunai SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pilih data pendaftar, klik **Aksi**.
5. Kemudian klik **Lihat Invoice**.
6. Sistem akan menampilkan daftar pembayaran milik data pendaftar yang dipilih.
7. Klik **Aksi**, klik **Konfirmasi Pembayaran**.
8. Sistem akan menampilkan popup konfirmasi, klik **Konfirmasi**.
9. Sistem akan mengubah status pembayaran tunai.

## Ubah Metode Pembayaran

Admin memiliki kemampuan untuk mengubah metode pembayaran:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pilih data pendaftar, klik **Aksi**.
5. Kemudian klik **Lihat Invoice**.
6. Sistem akan menampilkan daftar pembayaran milik data pendaftar yang dipilih.
7. Klik **Aksi**, klik **Ubah Metode Pembayaran**.
8. Sistem akan menampilkan konfirmasi ubah metode bayar, klik **Konfirmasi**.
9. Sistem akan mengubah metode pembayaran.

## Ubah Status

Admin memiliki kemampuan untuk mengubah status data pendaftar:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pilih data pendaftar yang ingin diubah statusnya melalui checkbox.
5. Pilih **Ubah Status**.
6. Isi form ubah status sesuai kebutuhan.
7. Klik **Simpan** untuk menyimpan pembaruan.

Status yang tersedia dapat ditambah melalui [Status Kustom pada Pengaturan Web](/cards-school/spmb-pengaturan-web#status-kustom).

## Hapus Calon Peserta Didik

Admin memiliki kemampuan untuk menghapus data pendaftar:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Pilih data pendaftar yang ingin dihapus melalui checkbox.
5. Pilih **Hapus Calon Peserta Didik**.
6. Sistem akan menampilkan konfirmasi hapus peserta didik, klik **Hapus**.

## Daftar Ulang

Admin memiliki kemampuan untuk mengubah status data pendaftar menjadi daftar ulang:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Data Pendaftar**.
4. Klik **Aksi** dan pilih **Daftar Ulang**.
5. Opsi Daftar Ulang hanya ditampilkan jika status data pendaftar telah **Diterima**.
6. Sistem akan menampilkan konfirmasi daftar ulang peserta didik, klik **Konfirmasi**.
7. Peserta didik akan menerima **Nomor Kartu** — lihat [Kartu](/referensi/glosarium#kartu).

Tagihan daftar ulang yang dibebankan otomatis diatur pada [Pengaturan Web — Daftar Ulang](/cards-school/spmb-pengaturan-web#daftar-ulang).

## Landing Page SPMB (Sisi Calon Pendaftar)

Landing Page SPMB memungkinkan calon pendaftar SPMB mengakses informasi, alur, dan proses pendaftaran peserta didik baru secara online. Langkah-langkahnya sebagai berikut:

1. Calon pendaftar mengakses website SPMB sekolah.
2. Sistem otomatis menampilkan halaman Landing Page SPMB yang memuat:
   - Alur pendaftaran SPMB.
   - Jadwal & biaya SPMB.
   - Persyaratan pendaftaran.
   - Informasi pendaftaran.
   - Daftar siswa diterima.
   - Formulir pendaftaran SPMB.
   - Cek pendaftaran SPMB.

## Formulir Pendaftaran

Formulir Pendaftaran SPMB memungkinkan calon peserta didik mengisi dan mengirimkan data pendaftaran sesuai dengan persyaratan yang ditentukan oleh sekolah. Langkah-langkahnya sebagai berikut:

1. Calon pendaftar mengakses website SPMB sekolah.
2. Sistem otomatis menampilkan halaman Landing Page SPMB.
3. Klik tombol **Daftar Sekarang**.
4. Sistem menampilkan halaman Pendaftaran yang dilengkapi dengan **Formulir Pendaftaran**.

Isi formulir ini ditentukan oleh konfigurasi [Formulir Standar](/cards-school/spmb-pengaturan-web#formulir-standar) dan [Formulir Kustom](/cards-school/spmb-pengaturan-web#formulir-kustom).

## Cek Pendaftaran

Cek Pendaftaran SPMB memungkinkan calon peserta didik memantau status dan informasi pendaftaran yang telah diajukan. Langkah-langkahnya sebagai berikut:

1. Calon pendaftar mengakses website SPMB sekolah.
2. Sistem otomatis menampilkan halaman Landing Page SPMB.
3. Klik tombol **Cek Pendaftaran**.
4. Masukkan **Nomor Pendaftaran**, klik **Cek Pendaftaran**.
5. Sistem akan menampilkan status dan informasi pendaftaran.
