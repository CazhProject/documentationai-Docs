Menu Kalender Akademik pada modul Akademik digunakan untuk mengelola event sekolah, seperti hari libur dan kegiatan akademik, per [instansi](/referensi/glosarium#instansi) dan [tahun ajaran](/referensi/glosarium#tahun-ajaran).

## List Event

Fitur ini digunakan untuk melihat list data event kalender akademik dengan cara berikut:

1. Login ke website.
2. Kemudian klik tab **Akademik**.
3. Kemudian klik **Kalender Akademik**.
4. Sistem akan menampilkan list data event kalender akademik yang telah ditambahkan.
5. Pada halaman ini user dapat mengelola **Tambah**, **Edit**, dan **Hapus** data event.

## Hapus Event

Fitur ini digunakan pengguna untuk menghapus event akademik yang tidak digunakan, dengan cara berikut:

1. Login ke website.
2. Kemudian klik tab **Akademik**.
3. Kemudian klik **Kalender Akademik**.
4. Sistem akan menampilkan **Daftar Event Akademik**.
5. Klik ikon **Hapus** pada baris data yang ingin dihapus.
6. Konfirmasi dengan klik **Hapus**.

## Edit Event

Fitur ini digunakan pengguna untuk memperbarui data event akademik yang kurang sesuai, dengan cara berikut:

1. Login ke website.
2. Kemudian klik tab **Akademik**.
3. Kemudian klik **Kalender Akademik**.
4. Sistem akan menampilkan **Daftar Event Akademik**.
5. Klik ikon **Edit**.
6. Sesuaikan informasi yang ingin Anda edit, lalu klik **Simpan**.

## Lihat Kalender

Fitur ini digunakan untuk melihat kalender akademik dengan cara berikut:

1. Login ke website.
2. Kemudian klik tab **Akademik**.
3. Kemudian klik **Kalender Akademik**.
4. Klik **Lihat Kalender Akademik**.
5. Pengguna dapat menampilkan kalender akademik berdasarkan **Instansi** dan **Tahun Ajaran**.
6. Sistem akan menampilkan kalender akademik yang memuat informasi mengenai hari libur nasional dan event yang telah ditambahkan.
