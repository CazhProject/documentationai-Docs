Menu Kalender Akademik pada modul Akademik digunakan admin sekolah untuk mengelola event sekolah, seperti hari libur dan kegiatan akademik, per [instansi](/referensi/glosarium#instansi) dan [tahun ajaran](/referensi/glosarium#tahun-ajaran).

## List Event

Lihat list data event kalender akademik yang telah ditambahkan.

<Steps>
  <Step title="Buka menu Akademik">Login ke website, kemudian klik tab **Akademik**.</Step>
  <Step title="Buka Kalender Akademik">Klik **Kalender Akademik**. Sistem menampilkan list data event kalender akademik yang telah ditambahkan.</Step>
  <Step title="Kelola event">Pada halaman ini pengguna dapat mengelola **Tambah**, **Edit**, dan **Hapus** data event.</Step>
</Steps>

## Hapus Event

Hapus event akademik yang tidak digunakan.

<Steps>
  <Step title="Buka Kalender Akademik">Login ke website, klik tab **Akademik**, lalu klik **Kalender Akademik**. Sistem menampilkan **Daftar Event Akademik**.</Step>
  <Step title="Hapus event">Klik ikon **Hapus** pada baris data yang ingin dihapus.</Step>
  <Step title="Konfirmasi">Konfirmasi dengan klik **Hapus**.</Step>
</Steps>

<Callout kind="alert">
Penghapusan event bersifat permanen dan event akan hilang dari tampilan kalender akademik.
</Callout>

## Edit Event

Perbarui data event akademik yang kurang sesuai.

<Steps>
  <Step title="Buka Kalender Akademik">Login ke website, klik tab **Akademik**, lalu klik **Kalender Akademik**. Sistem menampilkan **Daftar Event Akademik**.</Step>
  <Step title="Buka form edit">Klik ikon **Edit**.</Step>
  <Step title="Sesuaikan dan simpan">Sesuaikan informasi yang ingin Anda edit, lalu klik **Simpan**.</Step>
</Steps>

## Lihat Kalender

Tampilkan kalender akademik lengkap dengan hari libur nasional dan event yang telah ditambahkan.

<Steps>
  <Step title="Buka Kalender Akademik">Login ke website, klik tab **Akademik**, lalu klik **Kalender Akademik**.</Step>
  <Step title="Buka tampilan kalender">Klik **Lihat Kalender Akademik**.</Step>
  <Step title="Pilih tampilan">Pengguna dapat menampilkan kalender akademik berdasarkan **Instansi** dan **Tahun Ajaran**.</Step>
  <Step title="Lihat kalender">Sistem menampilkan kalender akademik yang memuat informasi hari libur nasional dan event yang telah ditambahkan.</Step>
</Steps>
