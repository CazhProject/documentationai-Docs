Menu Buku Tamu digunakan admin sekolah untuk mencatat kunjungan tamu ke instansi — baik diinput manual maupun melalui formulir yang diakses tamu lewat QR Code. Menu ini diakses melalui **Administrasi** → menu **Buku Tamu**.

## List Buku Tamu

Melihat dan mengelola daftar buku tamu.

<Steps>
  <Step title="Buka Buku Tamu">Login ke website menggunakan akun Admin, klik menu **Administrasi**, lalu pilih menu **Buku Tamu**. Sistem akan menampilkan halaman buku tamu.</Step>
  <Step title="Kelola daftar">Pada halaman buku tamu, Admin dapat melihat list buku tamu, export buku tamu, dan hapus buku tamu.</Step>
</Steps>

## Tambah Buku Tamu

Menambahkan data buku tamu secara manual.

<Steps>
  <Step title="Buka Buku Tamu">Login ke website menggunakan akun Admin, klik menu **Administrasi**, lalu pilih menu **Buku Tamu**. Sistem akan menampilkan halaman buku tamu.</Step>
  <Step title="Buka form tambah">Klik tombol **Tambah Buku Tamu**, lalu isi formulir buku tamu sesuai kebutuhan.</Step>
  <Step title="Simpan">Klik tombol **Simpan** untuk menyimpan data buku tamu.</Step>
</Steps>

## Edit Buku Tamu

Mengubah data buku tamu yang sudah tercatat.

<Steps>
  <Step title="Buka Buku Tamu">Login ke website menggunakan akun Admin, klik menu **Administrasi**, lalu pilih menu **Buku Tamu**. Sistem akan menampilkan halaman buku tamu.</Step>
  <Step title="Buka form edit">Pilih data yang ingin diubah, lalu klik opsi **Edit** pada data buku tamu yang dipilih.</Step>
  <Step title="Ubah dan simpan">Ubah data sesuai kebutuhan, lalu klik tombol **Simpan** untuk menyimpan perubahan data.</Step>
</Steps>

## Halaman Formulir Buku Tamu

Memberikan akses formulir buku tamu kepada tamu melalui QR Code.

<Steps>
  <Step title="Unduh QR Code">Login ke website menggunakan akun Admin, buka menu **Administrasi**, pilih menu **Buku Tamu**, lalu klik tombol **Download QR**. Sistem akan mengunduh QR Code formulir buku tamu.</Step>
  <Step title="Scan QR Code">Lakukan scan pada QR Code menggunakan perangkat yang tersedia. Sistem akan membuka halaman formulir buku tamu secara otomatis.</Step>
  <Step title="Isi formulir">Lengkapi data pada formulir sesuai kebutuhan, lalu klik tombol **Simpan** untuk menyimpan data buku tamu.</Step>
</Steps>
