Menu Buku Tamu digunakan Admin untuk mencatat kunjungan tamu ke instansi, baik diinput manual maupun melalui formulir yang diakses tamu lewat QR Code.

## List Buku Tamu

Admin memiliki kemampuan untuk mengelola buku tamu. Langkah-langkah untuk mengelola buku tamu:

1. Login ke website menggunakan akun Admin.
2. Klik menu **Administrasi**.
3. Pilih menu **Buku Tamu**.
4. Sistem akan menampilkan halaman buku tamu.
5. Pada halaman buku tamu, Admin dapat:
   - Melihat list buku tamu.
   - Export buku tamu.
   - Hapus buku tamu.

## Edit Buku Tamu

Admin dapat mengubah data buku tamu melalui langkah berikut:

1. Login ke website menggunakan akun Admin.
2. Klik menu **Administrasi**.
3. Pilih menu **Buku Tamu**.
4. Sistem akan menampilkan halaman buku tamu.
5. Pilih data yang ingin diubah.
6. Klik opsi **Edit** pada data buku tamu yang dipilih.
7. Ubah data sesuai kebutuhan.
8. Klik tombol **Simpan** untuk menyimpan perubahan data.

## Tambah Buku Tamu

Admin dapat menambahkan data buku tamu melalui langkah berikut:

1. Login ke website menggunakan akun Admin.
2. Klik menu **Administrasi**.
3. Pilih menu **Buku Tamu**.
4. Sistem akan menampilkan halaman buku tamu.
5. Klik tombol **Tambah Buku Tamu**.
6. Isi formulir buku tamu sesuai kebutuhan.
7. Klik tombol **Simpan** untuk menyimpan data buku tamu.

## Halaman Formulir Buku Tamu

Fitur ini digunakan untuk mengakses formulir buku tamu melalui QR Code. Cara penggunaan:

1. Login ke website menggunakan akun Admin.
2. Buka menu **Administrasi**.
3. Pilih menu **Buku Tamu**.
4. Pada halaman buku tamu, klik tombol **Download QR**.
5. Sistem akan mengunduh QR Code formulir buku tamu.
6. Lakukan scan pada QR Code menggunakan perangkat yang tersedia.
7. Sistem akan membuka halaman formulir buku tamu secara otomatis.
8. Lengkapi data pada formulir sesuai kebutuhan.
9. Klik tombol **Simpan** untuk menyimpan data buku tamu.
