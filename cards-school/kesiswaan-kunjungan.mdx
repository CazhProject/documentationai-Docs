Menu Kunjungan pada modul Kesiswaan digunakan admin untuk mengelola dan memantau data kunjungan yang tercatat dalam sistem, termasuk memproses status kunjungan menjadi Diterima atau Ditolak. Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Kunjungan

Fitur List Kunjungan digunakan untuk mengelola dan memantau data kunjungan yang tercatat dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kunjungan.
4. Pada halaman Kunjungan List, admin dapat:
   - Melihat daftar data kunjungan yang tercatat dalam sistem.
   - Melakukan pencarian data kunjungan menggunakan fitur **Search**.
   - Mengekspor data kunjungan menggunakan fitur **Export**.
   - Memantau dan menelusuri informasi kunjungan yang tersedia dalam sistem.

## Tambah Kunjungan

Fitur Tambah Kunjungan digunakan untuk menambahkan data kunjungan baru ke dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kunjungan.
4. Klik tombol **Tambah Kunjungan**.
5. Lengkapi form dengan sesuai dan benar, lalu klik tombol **Simpan**.

## Edit Kunjungan

Fitur Edit Kunjungan digunakan untuk memperbarui data kunjungan yang telah ada, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kunjungan.
4. Geser tabel ke kanan lalu klik **Titik Tiga**.
5. Pilih **Edit**.
6. Sistem menampilkan form detail data kunjungan yang dipilih.
7. Perbarui form dengan sesuai, lalu klik tombol **Simpan**.

## Hapus Kunjungan

Fitur Hapus Kunjungan memungkinkan admin menghapus baris data yang tidak valid, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem akan menampilkan halaman Kunjungan.
4. Pilih baris yang ingin dihapus dengan cara mencentang kolom paling kiri.
5. Kemudian pilih **Hapus yang Dipilih** pada bagian atas tabel.
6. Klik konfirmasi dan baris akan terhapus.

## Terima Kunjungan

Fitur Status Diterima digunakan untuk menetapkan atau memperbarui status data kunjungan menjadi **Diterima**, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem menampilkan Ringkasan Data Kunjungan.
4. Geser tabel ke kanan lalu klik **Titik Tiga**.
5. Pilih **Diterima**.

## Tolak Kunjungan

Fitur Status Ditolak digunakan untuk menetapkan atau memperbarui status data kunjungan menjadi **Ditolak**, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kunjungan**.
3. Sistem akan menampilkan halaman Data Kunjungan.
4. Klik **Titik Tiga** yang berada pada sebelah kanan tabel.
5. Kemudian pilih **Ditolak**.
6. Klik **Simpan** untuk menyimpan perubahan status.
