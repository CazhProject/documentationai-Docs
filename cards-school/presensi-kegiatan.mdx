Menu Presensi Kegiatan pada modul Akademik digunakan admin untuk mengelola kehadiran pada kegiatan sekolah (misalnya ekstrakurikuler atau acara khusus) untuk siswa, guru, dan staff. Kehadiran kegiatan siswa juga dapat dipantau orang tua — lihat [Akademik di CARDS Parent](/cards-parent/akademik).

## Log Presensi Kegiatan

Admin dapat mengakses data log presensi kegiatan untuk setiap siswa, guru, dan staff melalui langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Log**.
4. Sistem akan menampilkan **Data Log Presensi Kegiatan**.

### Edit Log Presensi Kegiatan

Admin dapat mengubah data log presensi kegiatan untuk setiap siswa, guru, dan staff melalui langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Log**.
4. Sistem akan menampilkan **Data Log Presensi Kegiatan**.
5. Klik tombol **Edit**.
6. Sistem menampilkan **Form Edit**.
7. Isi form edit log presensi kegiatan.
8. Klik **Simpan**.
9. Sistem akan menyimpan data.

## Rekap Kegiatan

Admin dapat melihat data rekap kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Rekap**.
4. Klik submenu **Kegiatan**.
5. Sistem akan menampilkan list data rekap kegiatan.

### Detail Rekap Kegiatan

Admin dapat melihat data detail rekap kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Rekap**.
4. Klik submenu **Kegiatan**.
5. Sistem akan menampilkan list data rekap kegiatan.
6. Klik tombol **>** pada data rekap kegiatan.
7. Sistem menampilkan detail rekap kegiatan yang dipilih.

## Pengaturan Presensi Kegiatan

Admin dapat melihat dan mengakses pengaturan presensi kegiatan yang telah dibuat, dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Pengaturan**.
4. Sistem menampilkan list pengaturan kegiatan yang telah dibuat.

### Edit Pengaturan Kegiatan

Admin dapat mengubah pengaturan kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Pengaturan**.
4. Sistem menampilkan list pengaturan kegiatan yang telah dibuat.
5. Klik tombol **Titik Tiga**.
6. Klik tombol **Edit**.
7. Isi/ubah form yang sudah terisi pada halaman Edit.
8. Klik tombol **Simpan**.

### Hapus Pengaturan Kegiatan

Admin dapat menghapus pengaturan kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Pengaturan**.
4. Sistem menampilkan list pengaturan kegiatan yang telah dibuat.
5. Klik tombol **Titik Tiga**.
6. Klik tombol **Hapus**.
7. Sistem menampilkan popup konfirmasi, klik tombol **Hapus**.

### Tambah Pengaturan Kegiatan

Admin dapat menambahkan pengaturan kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Pengaturan**.
4. Klik tombol **Tambah Kegiatan**.
5. Isi form **Tambah Kegiatan**.
6. Klik tombol **Simpan**.

**Catatan (improvement):** Pada form Tambah Kegiatan, field instansi dapat diisi lebih dari satu [instansi](/referensi/glosarium#instansi), dan terdapat field **tag** yang dapat digunakan jika ingin membuat kegiatan hanya dengan data tertentu.

## Link Presensi Kegiatan

Admin dapat mengelola link presensi kegiatan dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Kegiatan**.
3. Klik submenu **Pengaturan**.
4. Sistem akan menampilkan daftar presensi kegiatan.
5. Klik menu **Link Kegiatan** pada nama kegiatan.
6. Sistem akan redirect ke School Management System.
7. Lakukan presensi anggota dengan salah satu cara berikut:
   - Masukkan nomor kartu atau [RFID](/referensi/glosarium#rfid) pada menu tersebut.
   - Tampilkan QR code untuk discan anggota yang akan presensi (lihat [QR Presensi di CARDS Edu](/cards-edu/qr)).
   - Gunakan kamera perangkat untuk scan kartu anggota.
8. Pilih data anggota yang muncul; anggota otomatis akan terdaftar telah melakukan presensi.
