Menu Kesehatan pada modul Kesiswaan digunakan admin untuk mengelola dan memantau data kesehatan siswa yang tercatat dalam sistem. Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Kesehatan

Fitur List Kesehatan digunakan untuk mengelola dan memantau data kesehatan yang tercatat dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kesehatan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kesehatan.
4. Pada halaman Kesehatan List, admin dapat:
   - Melihat daftar data kesehatan yang tercatat dalam sistem.
   - Melakukan pencarian data kesehatan menggunakan fitur **Search**.
   - Mengekspor data kesehatan menggunakan fitur **Export**.
   - Memantau dan menelusuri informasi kesehatan yang tersedia dalam sistem.

## Tambah Kesehatan

Fitur Tambah Kesehatan digunakan untuk menambahkan data kesehatan baru ke dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kesehatan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kesehatan.
4. Klik **Tambah Kesehatan**.
5. Isi form dengan sesuai dan benar, lalu klik **Simpan**.

## Edit Kesehatan

Edit Kesehatan digunakan admin untuk mengedit atau memperbaiki data kesehatan, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kesehatan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kesehatan.
4. Klik ikon **Edit**.
5. Edit form dengan sesuai dan benar, lalu klik **Simpan**.

## Hapus Kesehatan

Hapus Kesehatan digunakan admin untuk menghapus data kesehatan, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Kesehatan**.
3. Sistem akan menampilkan halaman Ringkasan Data Kesehatan.
4. Klik ikon **Hapus**.
5. Konfirmasi penghapusan, lalu klik **Hapus**.
