Menu Prestasi pada modul Kesiswaan digunakan admin untuk mengelola pencatatan prestasi siswa. Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Prestasi

Admin memiliki kemampuan untuk mengelola data prestasi siswa melalui langkah berikut:

1. Login menggunakan akun Admin.
2. Klik menu **Kesiswaan**.
3. Pilih submenu **Prestasi**.
4. Sistem akan menampilkan halaman prestasi siswa.
5. Pada halaman tersebut, admin dapat:
   - Melihat data prestasi siswa.
   - Melakukan pencarian data prestasi.
   - Mengekspor data prestasi siswa.

## Tambah Prestasi

Admin dapat menambahkan informasi prestasi siswa ke dalam sistem melalui menu Prestasi.

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Prestasi**.
4. Sistem akan menampilkan halaman prestasi siswa.
5. Klik tombol **Tambah Prestasi**.
6. Lengkapi formulir prestasi siswa sesuai kebutuhan.
7. Klik tombol **Simpan** untuk menyimpan data prestasi siswa.

## Edit Prestasi

Admin dapat memperbarui informasi prestasi siswa melalui menu Prestasi.

1. Login menggunakan akun Admin.
2. Masuk ke menu **Kesiswaan**.
3. Pilih submenu **Prestasi**.
4. Sistem akan menampilkan halaman prestasi siswa.
5. Tentukan data prestasi yang akan diperbarui.
6. Klik tombol **Edit Prestasi**.
7. Ubah data prestasi sesuai kebutuhan.
8. Klik tombol **Simpan** untuk menyimpan perubahan data prestasi siswa.

## Hapus Prestasi

Admin dapat menghapus data prestasi siswa melalui menu Prestasi.

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Prestasi**.
4. Sistem akan menampilkan halaman prestasi siswa.
5. Pilih data prestasi yang ingin dihapus.
6. Klik tombol **Hapus Prestasi**.
7. Klik tombol **Hapus** untuk mengonfirmasi proses penghapusan data prestasi.
