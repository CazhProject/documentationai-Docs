Menu Prestasi pada modul Kesiswaan digunakan admin sekolah untuk mengelola pencatatan prestasi siswa.

<Callout kind="info">
Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).
</Callout>

## List Prestasi

Lihat, cari, dan ekspor data prestasi siswa.

<Steps>
  <Step title="Buka menu Kesiswaan">Login menggunakan akun Admin, lalu klik menu **Kesiswaan**.</Step>
  <Step title="Buka Prestasi">Pilih submenu **Prestasi**. Sistem menampilkan halaman prestasi siswa.</Step>
  <Step title="Kelola daftar">
    Pada halaman tersebut, admin dapat:
    - Melihat data prestasi siswa.
    - Melakukan pencarian data prestasi.
    - Mengekspor data prestasi siswa.
  </Step>
</Steps>

## Tambah Prestasi

Catat prestasi siswa baru ke dalam sistem.

<Steps>
  <Step title="Buka Prestasi">Login menggunakan akun Admin, buka menu **Kesiswaan**, lalu pilih submenu **Prestasi**. Sistem menampilkan halaman prestasi siswa.</Step>
  <Step title="Buka form tambah">Klik tombol **Tambah Prestasi**.</Step>
  <Step title="Isi formulir">Lengkapi formulir prestasi siswa sesuai kebutuhan.</Step>
  <Step title="Simpan">Klik tombol **Simpan** untuk menyimpan data prestasi siswa.</Step>
</Steps>

## Edit Prestasi

Perbarui informasi prestasi siswa yang sudah tercatat.

<Steps>
  <Step title="Buka Prestasi">Login menggunakan akun Admin, masuk ke menu **Kesiswaan**, lalu pilih submenu **Prestasi**. Sistem menampilkan halaman prestasi siswa.</Step>
  <Step title="Pilih data">Tentukan data prestasi yang akan diperbarui, lalu klik tombol **Edit Prestasi**.</Step>
  <Step title="Ubah data">Ubah data prestasi sesuai kebutuhan.</Step>
  <Step title="Simpan">Klik tombol **Simpan** untuk menyimpan perubahan data prestasi siswa.</Step>
</Steps>

## Hapus Prestasi

Hapus data prestasi siswa dari sistem.

<Steps>
  <Step title="Buka Prestasi">Login menggunakan akun Admin, buka menu **Kesiswaan**, lalu pilih submenu **Prestasi**. Sistem menampilkan halaman prestasi siswa.</Step>
  <Step title="Pilih data">Pilih data prestasi yang ingin dihapus, lalu klik tombol **Hapus Prestasi**.</Step>
  <Step title="Konfirmasi">Klik tombol **Hapus** untuk mengonfirmasi proses penghapusan data prestasi.</Step>
</Steps>

<Callout kind="alert">
Penghapusan data prestasi bersifat permanen dan tidak dapat dikembalikan.
</Callout>
