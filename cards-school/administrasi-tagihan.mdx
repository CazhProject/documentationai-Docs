Menu Tagihan digunakan Admin untuk membuat dan memonitor seluruh tagihan anggota (siswa, guru, dan staf), memantau pembayaran, serta menindaklanjuti tunggakan. Untuk pembayaran tagihan oleh Admin, lihat [Keuangan: Transaksi](/cards-school/keuangan-transaksi). Orang tua juga dapat membayar tagihan langsung dari aplikasi — lihat panduan [CARDS Parent](/cards-parent/tagihan).

## Daftar Data Tagihan

Admin memiliki kemampuan untuk memonitor data tagihan anggota. Langkah untuk akses data tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Data Tagihan**.
4. Sistem akan menampilkan halaman Data Tagihan.
5. Pada halaman data tagihan, Admin dapat:
   - Memonitor data tagihan seluruh anggota.
   - Melihat ringkasan nominal tagihan terbayar dan belum terbayar.
   - Export data tagihan.

## Edit Data Tagihan

Admin memiliki kemampuan untuk memperbarui data tagihan anggota. Langkah untuk memperbarui data tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Data Tagihan**.
4. Sistem akan menampilkan halaman Data Tagihan.
5. Pilih salah satu data tagihan; gunakan fitur pencarian atau filter untuk menemukan data tagihan secara spesifik.
6. Klik pada **Nama** atau klik ikon **Edit** pada baris data tagihan yang dipilih.
7. Sistem akan menampilkan halaman detail tagihan yang dipilih.
8. Pada halaman detail data tagihan, Admin dapat:
   - Edit total tagihan.
   - Edit catatan.
   - Melihat riwayat pembayaran tagihan.
   - Edit nominal total terbayar untuk metode pembayaran tunai.

## Hapus Data Tagihan

Admin memiliki kemampuan untuk menghapus data tagihan belum terbayar milik seluruh anggota. Langkah untuk hapus data tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Data Tagihan**.
4. Sistem akan menampilkan halaman Data Tagihan.
5. Hapus tagihan belum terbayar dapat dilakukan secara massal maupun satuan:
   - **Hapus tagihan belum terbayar satuan:**
     1. Pilih salah satu data tagihan; gunakan fitur pencarian atau filter untuk menemukan data tagihan secara spesifik.
     2. Klik ikon **Hapus** pada baris data tagihan yang dipilih.
   - **Hapus tagihan belum terbayar massal:**
     1. Pilih beberapa data tagihan melalui checkbox.
     2. Klik **Hapus**.
6. Sistem akan menampilkan popup konfirmasi hapus data tagihan.

## Buat Tagihan Satuan

Admin memiliki kemampuan untuk membuat data tagihan anggota secara satuan. Langkah untuk buat data tagihan satuan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Buat/Import Tagihan**.
4. Sistem akan menampilkan halaman Formulir Buat Tagihan Satuan.
5. Lengkapi formulir buat tagihan. Admin dapat:
   - Mencari data anggota berdasarkan Nama atau Nomor Kartu.
   - Membuat tagihan berdasarkan Jenis Tagihan.
   - Mengisi nominal Jumlah Tagihan sesuai kebutuhan.
   - Membuat tagihan dengan jenis Statis VA maupun Dinamis VA.

## Buat Tagihan Massal

Admin memiliki kemampuan untuk membuat data tagihan anggota secara massal. Langkah untuk buat data tagihan massal:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Buat/Import Tagihan**.
4. Klik tab **Massal**.
5. Sistem akan menampilkan halaman Formulir Buat Tagihan Massal.
6. Lengkapi formulir buat tagihan. Admin dapat:
   - Memilih penerima tagihan berdasarkan Jenis Anggota (siswa, guru, dan staf).
   - Memilih penerima tagihan secara spesifik berdasarkan Tag.
   - Membuat tagihan berdasarkan Jenis Tagihan.
   - Mengisi nominal Jumlah Tagihan sesuai kebutuhan.
   - Membuat tagihan dengan jenis Statis VA maupun Dinamis VA.

## Import Data Tagihan

Admin memiliki kemampuan untuk melakukan import data tagihan. Langkah untuk import data tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Buat/Import Tagihan**.
4. Klik tab **Import**.
5. Sistem akan menampilkan halaman Formulir Import Tagihan.
6. Lengkapi formulir import tagihan. Admin dapat:
   - Mengunduh template import tagihan.
   - Menginput nominal tagihan untuk masing-masing anggota penerima tagihan pada file template.
   - Mengunggah file import tagihan.

## Laporan Pembayaran Tagihan

Admin memiliki kemampuan untuk memonitor laporan pembayaran tagihan seluruh anggota melalui sistem. Langkah untuk akses laporan pembayaran tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Laporan Pembayaran**.
4. Sistem akan menampilkan halaman Laporan Pembayaran.
5. Pada halaman laporan pembayaran, Admin dapat:
   - Memonitor laporan pembayaran seluruh anggota.
   - Mencetak struk bukti pembayaran.
   - Melihat ringkasan total penerimaan tagihan (tunai dan non-tunai).

## Daftar Rekap Pembayaran Tagihan

Admin memiliki kemampuan untuk memonitor rekap pembayaran setiap jenis tagihan melalui sistem. Langkah untuk akses daftar rekap pembayaran tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Rekap Pembayaran**.
4. Sistem akan menampilkan halaman Daftar Rekap Pembayaran.
5. Pada halaman tersebut, Admin dapat memonitor daftar rekap pembayaran dan total penerimaan setiap jenis tagihan.

## Detail Rekap Pembayaran Tagihan

Langkah untuk akses detail rekap pembayaran tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Rekap Pembayaran**.
4. Sistem akan menampilkan halaman Daftar Rekap Pembayaran.
5. Pilih salah satu data rekap pembayaran; gunakan fitur pencarian atau filter untuk menemukan jenis tagihan secara spesifik.
6. Klik ikon **Detail** pada baris rekap pembayaran yang dipilih.
7. Sistem akan menampilkan halaman Detail Rekap Pembayaran untuk jenis tagihan tersebut. Admin dapat:
   - Melihat ringkasan total penerimaan untuk jenis tagihan yang dipilih.
   - Melihat daftar pembayaran.
   - Melihat dan mencetak bukti pembayaran setiap daftar pembayaran tagihan.
   - Export daftar pembayaran.
   - Filter daftar pembayaran berdasarkan metode pembayaran.

## Tunggakan Tagihan

Admin memiliki kemampuan untuk memonitor tunggakan tagihan seluruh anggota. Langkah untuk akses tunggakan tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Tunggakan**.
4. Sistem akan menampilkan halaman Daftar Tunggakan.
5. Pada halaman tersebut, Admin dapat:
   - Memonitor daftar tunggakan seluruh anggota.
   - Menampilkan daftar tunggakan secara spesifik menggunakan filter maupun pencarian.
   - Melakukan export daftar tunggakan.

## Detail Tunggakan Tagihan

Langkah untuk akses detail tunggakan tagihan:

1. Login ke website, lalu klik menu **Administrasi**.
2. Kemudian, klik submenu **Tagihan**.
3. Klik **Tunggakan**.
4. Sistem akan menampilkan halaman Daftar Tunggakan.
5. Pilih salah satu data tunggakan; gunakan fitur pencarian atau filter untuk menemukan anggota secara spesifik.
6. Klik ikon **Detail** pada baris data tunggakan yang dipilih.
7. Sistem akan menampilkan halaman Detail Tunggakan untuk anggota tersebut. Admin dapat:
   - Melihat daftar tagihan.
   - Membuat invoice tunggakan.
   - Membayar tagihan menggunakan [Kartu](/referensi/glosarium#kartu).
   - Membayar tagihan menggunakan Tunai.
