Menu Pengguna pada modul Pengaturan digunakan admin sekolah untuk mengelola [jenis pengguna](/referensi/glosarium#jenis-pengguna) (role) dan akun pengguna dalam sistem CARDS School.

## Jenis Pengguna

### List Jenis Pengguna

Halaman **List Jenis Pengguna** digunakan untuk melihat daftar jenis pengguna (role) yang tersedia pada sistem beserta ringkasan hak akses dan jumlah pengguna yang menggunakan role tersebut. Melalui halaman ini, Admin juga dapat mengakses fitur **Lihat Pengguna** dan **Edit** pada setiap jenis pengguna.

</Callout>

**Informasi yang Ditampilkan**

Pada setiap kartu **Jenis Pengguna**, sistem menampilkan informasi berikut:

| Informasi           | Keterangan                                                        |
| ------------------- | ----------------------------------------------------------------- |
| Nama Jenis Pengguna | Nama role atau jenis pengguna.                                    |
| Total Pengguna      | Jumlah akun yang menggunakan jenis pengguna tersebut.             |
| Hak Akses           | Ringkasan menu atau fitur yang dapat diakses oleh jenis pengguna. |
| Lihat Pengguna      | Membuka daftar akun yang menggunakan jenis pengguna tersebut.     |
| Edit                | Mengubah nama atau hak akses jenis pengguna.                      |

<Callout kind="tip" collapsed="false">
  Apabila jumlah hak akses cukup banyak, sistem akan menampilkan indikator tambahan (misalnya +1, +2, dan seterusnya) untuk menunjukkan masih ada hak akses lain yang dimiliki.
</Callout>

### Tambah Jenis Pengguna

Halaman **Tambah Jenis Pengguna** digunakan untuk membuat jenis pengguna (role) baru beserta hak akses yang dimiliki. Setiap jenis pengguna dapat diberikan akses ke menu dan fitur tertentu sesuai dengan kebutuhan.

[https://drive.google.com/file/d/12OlGfeGDHhGfaq4AbS2\_xBhtVy-aoDlk/view?usp=drive\_link](https://drive.google.com/file/d/12OlGfeGDHhGfaq4AbS2_xBhtVy-aoDlk/view?usp=drive_link)

<Callout kind="info" collapsed="false">
  Pastikan Anda sudah login sebagai **Admin** dan memiliki hak akses ke menu **Pengaturan → Pengguna → Jenis Pengguna**.
</Callout>

<Steps>
  <Step title="Buka menu Jenis Pengguna" title-type="p">
    Pada sidebar, klik **Pengaturan**, kemudian pilih **Pengguna** dan buka menu **Jenis Pengguna**.

    <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784192677056-wzrkay5t5u7-tambah-jenis-pengguna-1.png?fm=auto&auto=compress%2Cformat" width="2559" height="1273" alt="" />
  </Step>

  <Step title="Klik Tambah Jenis Pengguna" title-type="p">

    <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784192722824-c9xt7yc0yfb-tambah-jenis-pengguna-2.png?fm=auto&auto=compress%2Cformat" width="2559" height="1274" alt="" />

    <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784192759643-r7f23s1ved-tambah-jenis-pengguna-3.png?fm=auto&auto=compress%2Cformat" width="2559" height="1270" alt="" />

### Isian formulir

| Isian               | Ketentuan                         |
| ------------------- | --------------------------------- |
| Nama Jenis Pengguna | Wajib, maksimal 255 karakter.     |
| Akses Menu          | Wajib, minimal satu menu dipilih. |

<Callout kind="info" collapsed="false">
  Hak akses disusun dua tingkat: menu utama (Anggota, Administrasi, Keuangan, Kartu, Akademik, Pengaturan, LMS) dan submenu di bawahnya (misalnya Administrasi memiliki submenu Tagihan, Inventaris, dan Pengumuman; Keuangan memiliki submenu Pembukuan dan Tabungan). Mencentang menu induk otomatis memilih seluruh submenunya.
</Callout>

### Edit Jenis Pengguna

Halaman **Edit Jenis Pengguna** digunakan untuk mengubah informasi dan hak akses dari jenis pengguna yang telah dibuat. Admin dapat memperbarui nama jenis pengguna, status, pengaturan batas akses data, serta hak akses pada setiap menu dan fitur.

<Callout kind="info" collapsed="false">
  Pastikan Anda sudah login sebagai **Admin** dan memiliki hak akses ke menu **Pengaturan → Pengguna → Jenis Pengguna**.
</Callout>

<Steps>
  <Step title="Buka menu Jenis Pengguna" title-type="p">
    Pada sidebar, klik **Pengaturan**, kemudian pilih **Pengguna** dan buka menu **Jenis Pengguna**.
  </Step>

  <Step title="Pilih jenis pengguna yang akan diedit" title-type="p">
    Pada daftar **Jenis Pengguna**, cari jenis pengguna yang ingin diubah, kemudian klik tombol **Edit**.
  </Step>

  <Step title="Ubah informasi jenis pengguna" title-type="p">
    Sistem akan menampilkan halaman **Edit Jenis Pengguna**.

    Lakukan perubahan sesuai kebutuhan, seperti:

    - Nama Jenis Pengguna.
    - Status (Aktif atau Nonaktif).
    - Pengaturan **Batasi Akses Data**.
  </Step>

  <Step title="Perbarui hak akses" title-type="p">
    Pilih kategori menu yang ingin diubah, seperti:

    - Anggota
    - Administrasi
    - Keuangan
    - Kartu
    - Akademik
    - Kesiswaan
    - Pengaturan
    - LMS

    Kemudian ubah hak akses pada setiap fitur sesuai kebutuhan.
  </Step>

  <Step title="Simpan perubahan" title-type="p">
    Setelah seluruh perubahan selesai dilakukan, klik tombol **Simpan**.

    Apabila data berhasil diperbarui, sistem akan menampilkan notifikasi bahwa **Jenis Pengguna berhasil diubah** dan perubahan akan langsung diterapkan.
  </Step>
</Steps>

### Lihat Pengguna pada Jenis Pengguna

Lihat akun-akun yang ada pada suatu jenis pengguna.

## Akun Pengguna

### List Akun Pengguna

Lihat, filter, cari, dan ekspor daftar akun pengguna.

Cara menggunakan Filter, Pencarian, dan Export:

| Fitur         | Cara pakai                                                   |
| :------------ | :----------------------------------------------------------- |
| **Filter**    | Klik submenu Filter → isi filter sesuai kebutuhan.           |
| **Pencarian** | Klik menu Cari → input akun yang ingin dicari.               |
| **Export**    | Klik Excel → buka file Excel yang berisi data akun pengguna. |

### Edit Akun Pengguna

Ubah data akun pengguna.

<Callout kind="info" collapsed="false">
  Pada edit akun, **email dan nomor HP tidak dapat diubah**; status akun wajib dipilih.
</Callout>

### Hapus Akun Pengguna

Hapus akun pengguna dari sistem. Aksi ini menghapus akun login pengguna sehingga akun tersebut tidak lagi tampil di daftar dan tidak dapat digunakan untuk masuk ke CARDS School. Biasanya dipakai saat pengguna sudah tidak lagi bertugas atau akun salah dibuat.

<Callout kind="alert" collapsed="false">
  Penghapusan akun pengguna bersifat permanen. Pastikan akun memang sudah tidak digunakan sebelum menghapus. Pengguna tidak dapat menghapus akunnya sendiri.
</Callout>

### Tambah Akun Pengguna

Buat akun pengguna baru.

### Isian formulir

| Isian                 | Ketentuan                              |
| --------------------- | -------------------------------------- |
| Instansi              | Wajib, dapat memilih lebih dari satu.  |
| Nama                  | Wajib, maksimal 255 karakter.          |
| Email                 | Wajib, format email, unik.             |
| Nomor HP              | Wajib, angka 8–15 digit, unik.         |
| Alamat                | Opsional, maksimal 255 karakter.       |
| Jenis Pengguna (Role) | Wajib.                                 |
| Jenis Akun            | Opsional (penanda akun partner/pusat). |

<Callout kind="info" collapsed="false">
  Saat akun dibuat, sistem otomatis membuat kata sandi acak (12 karakter) dan PIN 6 digit, lalu mendaftarkan akun ke layanan autentikasi Cazh.
</Callout>
