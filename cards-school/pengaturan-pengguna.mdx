Menu Pengguna pada modul Pengaturan digunakan admin untuk mengelola [jenis pengguna](/referensi/glosarium#jenis-pengguna) (role) dan akun pengguna dalam sistem CARDS School.

## Jenis Pengguna

### List Jenis Pengguna

Admin dapat melihat jenis pengguna. Langkah untuk melihat jenis pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Klik **Jenis Pengguna**.
4. Sistem akan menampilkan halaman Jenis Pengguna.
5. Pada halaman List Jenis Pengguna, admin dapat melihat jenis pengguna.

### Tambah Jenis Pengguna

Admin memiliki kemampuan untuk menambah jenis pengguna. Langkah untuk menambah jenis pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Lalu klik **Jenis Pengguna**.
4. Sistem akan menampilkan halaman Jenis Pengguna.
5. Lalu klik **Tambah Jenis Pengguna**.
6. Isi form → klik **Simpan**.
7. Pada List Jenis Pengguna, admin dapat menambah jenis pengguna baru.

### Edit Jenis Pengguna

Admin memiliki kemampuan untuk mengedit jenis pengguna. Langkah untuk edit jenis pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Sistem akan menampilkan halaman Jenis Pengguna.
4. Lalu klik **Edit Jenis Pengguna**.
5. Pada List Jenis Pengguna, admin dapat mengedit jenis pengguna.

### Lihat Pengguna pada Jenis Pengguna

Admin memiliki kemampuan untuk melihat akun yang ada pada suatu jenis pengguna. Langkah untuk mengaksesnya:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Jenis Pengguna**.
3. Sistem akan menampilkan halaman Jenis Pengguna.
4. Lalu klik **Lihat Pengguna**.
5. Sistem akan menampilkan daftar akun pengguna.

## Akun Pengguna

### List Akun Pengguna

Admin memiliki kemampuan untuk melihat daftar akun pengguna. Langkah untuk mengaksesnya:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Akun Pengguna**.
3. Sistem akan menampilkan halaman Akun Pengguna.
4. Pada halaman List Akun Pengguna, admin dapat:
   - Melihat list akun pengguna.
   - Filter akun pengguna.
   - Export akun pengguna.
   - Mencari akun pengguna.
5. Cara menggunakan Filter, Pencarian, dan Export:
   - **Filter**: klik submenu Filter → isi filter sesuai kebutuhan.
   - **Pencarian**: klik menu Cari → input akun yang ingin dicari.
   - **Export**: klik Excel → buka file Excel yang berisi data akun pengguna.

### Edit Akun Pengguna

Admin memiliki kemampuan untuk mengedit akun pengguna. Langkah untuk edit akun pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Lalu klik **Akun Pengguna**.
4. Sistem akan menampilkan Akun Pengguna.
5. Klik **Edit Akun Pengguna** pada sisi kanan akun pengguna.
6. Edit data akun sesuai data akun pengguna → klik **Simpan**.

### Hapus Akun Pengguna

Admin memiliki kemampuan untuk menghapus akun pengguna. Langkah untuk hapus akun pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Lalu klik **Akun Pengguna**.
4. Sistem akan menampilkan halaman Akun Pengguna.
5. Klik **Hapus Pengguna** pada sisi kanan akun pengguna.

### Tambah Akun Pengguna

Admin memiliki kemampuan untuk menambah akun pengguna. Langkah untuk tambah akun pengguna:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Pengguna**.
3. Lalu klik **Akun Pengguna**.
4. Sistem akan menampilkan halaman Akun Pengguna.
5. Klik **Tambah Pengguna**.
6. Isi form sesuai data siswa → klik **Simpan**.
