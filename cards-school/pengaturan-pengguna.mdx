Menu Pengguna pada modul Pengaturan digunakan admin sekolah untuk mengelola [jenis pengguna](/referensi/glosarium#jenis-pengguna) (role) dan akun pengguna dalam sistem CARDS School.

## Jenis Pengguna

### List Jenis Pengguna

Lihat daftar jenis pengguna yang tersedia.

<Steps>
  <Step title="Buka menu Pengaturan">Login ke website, kemudian klik menu **Pengaturan**.</Step>
  <Step title="Buka Pengguna">Klik submenu **Pengguna**, lalu klik **Jenis Pengguna**. Sistem menampilkan halaman Jenis Pengguna.</Step>
  <Step title="Lihat daftar">Pada halaman List Jenis Pengguna, admin dapat melihat jenis pengguna.</Step>
</Steps>

### Tambah Jenis Pengguna

Buat jenis pengguna baru.

<Steps>
  <Step title="Buka Jenis Pengguna">Login ke website, klik menu **Pengaturan**, klik submenu **Pengguna**, lalu klik **Jenis Pengguna**. Sistem menampilkan halaman Jenis Pengguna.</Step>
  <Step title="Buka form tambah">Klik **Tambah Jenis Pengguna**.</Step>
  <Step title="Isi dan simpan">Isi form, lalu klik **Simpan**. Jenis pengguna baru muncul pada List Jenis Pengguna.</Step>
</Steps>

### Edit Jenis Pengguna

Ubah jenis pengguna yang sudah ada.

<Steps>
  <Step title="Buka menu Pengaturan">Login ke website, kemudian klik menu **Pengaturan**.</Step>
  <Step title="Buka Pengguna">Klik submenu **Pengguna**. Sistem menampilkan halaman Jenis Pengguna.</Step>
  <Step title="Edit">Klik **Edit Jenis Pengguna**. Pada List Jenis Pengguna, admin dapat mengedit jenis pengguna.</Step>
</Steps>

### Lihat Pengguna pada Jenis Pengguna

Lihat akun-akun yang ada pada suatu jenis pengguna.

<Steps>
  <Step title="Buka menu Pengaturan">Login ke website, kemudian klik menu **Pengaturan**.</Step>
  <Step title="Buka Jenis Pengguna">Klik submenu **Jenis Pengguna**. Sistem menampilkan halaman Jenis Pengguna.</Step>
  <Step title="Lihat pengguna">Klik **Lihat Pengguna**. Sistem menampilkan daftar akun pengguna.</Step>
</Steps>

## Akun Pengguna

### List Akun Pengguna

Lihat, filter, cari, dan ekspor daftar akun pengguna.

<Steps>
  <Step title="Buka menu Pengaturan">Login ke website, kemudian klik menu **Pengaturan**.</Step>
  <Step title="Buka Akun Pengguna">Klik submenu **Akun Pengguna**. Sistem menampilkan halaman Akun Pengguna.</Step>
  <Step title="Kelola daftar">Pada halaman List Akun Pengguna, admin dapat melihat list akun pengguna, filter, export, dan mencari akun pengguna.</Step>
</Steps>

Cara menggunakan Filter, Pencarian, dan Export:

| Fitur | Cara pakai |
| :--- | :--- |
| **Filter** | Klik submenu Filter → isi filter sesuai kebutuhan. |
| **Pencarian** | Klik menu Cari → input akun yang ingin dicari. |
| **Export** | Klik Excel → buka file Excel yang berisi data akun pengguna. |

### Edit Akun Pengguna

Ubah data akun pengguna.

<Steps>
  <Step title="Buka Akun Pengguna">Login ke website, klik menu **Pengaturan**, klik submenu **Pengguna**, lalu klik **Akun Pengguna**. Sistem menampilkan Akun Pengguna.</Step>
  <Step title="Buka form edit">Klik **Edit Akun Pengguna** pada sisi kanan akun pengguna.</Step>
  <Step title="Ubah dan simpan">Edit data akun sesuai data akun pengguna, lalu klik **Simpan**.</Step>
</Steps>

### Hapus Akun Pengguna

Hapus akun pengguna dari sistem.

<Steps>
  <Step title="Buka Akun Pengguna">Login ke website, klik menu **Pengaturan**, klik submenu **Pengguna**, lalu klik **Akun Pengguna**. Sistem menampilkan halaman Akun Pengguna.</Step>
  <Step title="Hapus akun">Klik **Hapus Pengguna** pada sisi kanan akun pengguna.</Step>
</Steps>

<Callout kind="alert">
Penghapusan akun pengguna bersifat permanen. Pastikan akun memang sudah tidak digunakan sebelum menghapus.
</Callout>

### Tambah Akun Pengguna

Buat akun pengguna baru.

<Steps>
  <Step title="Buka Akun Pengguna">Login ke website, klik menu **Pengaturan**, klik submenu **Pengguna**, lalu klik **Akun Pengguna**. Sistem menampilkan halaman Akun Pengguna.</Step>
  <Step title="Buka form tambah">Klik **Tambah Pengguna**.</Step>
  <Step title="Isi dan simpan">Isi form sesuai data siswa, lalu klik **Simpan**.</Step>
</Steps>
