Halaman ini merangkum panduan lengkap submenu Pengaturan lain di CARDS School: Kesiswaan, Lembaga, Kepegawaian, Tagihan, Aplikasi, dan Administrasi.

## Pengaturan Kesiswaan

### Waktu Perizinan

Admin memiliki kemampuan untuk melakukan pengaturan waktu perizinan. Langkah mengaturnya:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Kesiswaan**.
3. Kemudian klik tab **Waktu Perizinan**.
4. Sistem menampilkan form untuk memilih [instansi](/referensi/glosarium#instansi).
5. Isi form batas waktu maksimal.
6. User tidak bisa mengajukan izin melebihi batas maksimal waktu yang sudah ditentukan (lihat [Kesiswaan — Perizinan](/cards-school/kesiswaan-perizinan)).

### Tipe Pelanggaran

Fitur ini digunakan agar pengguna dapat menambahkan, mengedit, dan menghapus tipe pelanggaran siswa sesuai kebutuhan. Langkah penggunaan:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Kesiswaan**.
3. Kemudian klik tab **Tipe Pelanggaran**.
4. Sistem menampilkan data Tipe Pelanggaran yang sudah dibuat. Jika belum ada data, sistem akan menampilkan halaman kosong.
5. Kemudian user dapat menambahkan data Tipe Pelanggaran (digunakan pada [Kesiswaan — Pelanggaran](/cards-school/kesiswaan-pelanggaran)).

## Pengaturan Lembaga

Pengaturan Lembaga digunakan agar user dapat mengelola [lembaga](/referensi/glosarium#lembaga):

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Lembaga**.
3. Sistem akan menampilkan halaman Lembaga.
4. Pada halaman Lembaga dapat mengelola data identitas lembaga.

### Detail Lembaga

Detail Pengaturan Lembaga digunakan agar user dapat mengubah pengaturan yang sudah dibuat, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Lembaga**.
3. Sistem akan menampilkan halaman Lembaga.
4. Klik tombol **Action** pada list tabel.
5. Sistem menampilkan halaman Detail Lembaga.
6. Ubah data pada form di halaman detail lembaga.
7. Klik tombol **Simpan**.

## Pengaturan Tagihan

### Jenis Tagihan — List

List Jenis Tagihan memungkinkan admin melihat dan mengelola daftar [jenis tagihan](/referensi/glosarium#jenis-tagihan) yang tersedia dalam sistem. Langkah-langkahnya sebagai berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Tagihan**.
3. Klik/pilih submenu **Jenis Tagihan**.
4. Sistem menampilkan halaman Jenis Tagihan.
5. Sistem menampilkan list data Jenis Tagihan.
6. Pada halaman Jenis Tagihan admin dapat search & filter data.

### Jenis Tagihan — Edit

Edit Jenis Tagihan memungkinkan admin mengubah informasi dan pengaturan jenis tagihan yang telah dibuat. Langkah-langkahnya sebagai berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Tagihan**.
3. Klik/pilih submenu **Jenis Tagihan**.
4. Sistem menampilkan halaman Jenis Tagihan.
5. Sistem menampilkan list data Jenis Tagihan.
6. Klik tombol action **Edit** pada salah satu data.
7. Pada form edit, ubah data & klik tombol **Simpan**.

### Jenis Tagihan — Hapus

Hapus Jenis Tagihan memungkinkan admin menghapus jenis tagihan yang sudah tidak digunakan dari sistem. Langkah-langkahnya sebagai berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Tagihan**.
3. Klik/pilih submenu **Jenis Tagihan**.
4. Sistem menampilkan halaman Jenis Tagihan.
5. Sistem menampilkan list data Jenis Tagihan.
6. Klik tombol action **Hapus** pada salah satu data.
7. Sistem menampilkan popup Hapus, lalu klik tombol **Hapus**.

### Jenis Tagihan — Tambah

Tambah Jenis Tagihan memungkinkan admin menambahkan jenis tagihan baru yang dapat digunakan dalam pengelolaan tagihan pada sistem. Langkah-langkahnya sebagai berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Tagihan**.
3. Klik/pilih submenu **Jenis Tagihan**.
4. Sistem menampilkan halaman Jenis Tagihan.
5. Sistem menampilkan list data Jenis Tagihan.
6. Klik tombol **Tambah Jenis Tagihan**.
7. Pada form tambah, input data & klik tombol **Simpan**.

## Pengaturan Aplikasi

### Umum

Pengaturan aplikasi Umum digunakan agar user dapat mengelola nama aplikasi dan menyembunyikan footer jika diinginkan:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Aplikasi**.
3. Sistem akan menampilkan tab **Umum**.
4. User dapat mengelola **Pengaturan Nama Aplikasi**, **Penamaan Tagihan**, **Penamaan Siswa**, dan **Sembunyikan Footer**.

### Partner

Pengaturan aplikasi Partner digunakan agar user dapat mengelola identitas aplikasi dengan cara berikut:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Aplikasi**.
3. Kemudian klik tab **Partner**.
4. Sistem akan menampilkan form Partner.
5. User dapat mengelola **Identitas Partner**.

### Halaman Utama

Pengaturan aplikasi Halaman Utama digunakan agar user dapat mengelola banner dan sosial media aplikasi dengan cara berikut:

3. Kemudian klik tab **Halaman Utama**.

### Nama Program Penerimaan

Fitur ini digunakan agar pengguna dapat menentukan penamaan program penerimaan peserta didik baru sesuai kebutuhan. Langkah penggunaan:

1. Login ke website, kemudian klik menu **Pengaturan**.
2. Kemudian klik submenu **Aplikasi**.
3. Kemudian klik tab **Nama Program Penerimaan**.
4. Sistem akan menampilkan form Nama Program Penerimaan.

### Improvement Pengaturan Aplikasi

Admin memiliki kemampuan untuk meng-improve pengaturan aplikasi. Langkahnya:

1. Login ke website, lalu klik **Pengaturan**.
2. Pilih menu **Aplikasi**.
3. Klik navbar **Umum**.
4. Sistem akan menampilkan form yang bisa diedit.
5. Pada Aplikasi — Umum, admin dapat melakukan improvement pada pengaturan baru.

## Pengaturan Administrasi

### Kategori Pengumuman

Fitur ini digunakan untuk mengelola data kategori pengumuman dengan cara berikut:

1. Login ke website.
2. Kemudian klik menu **Pengaturan**.
3. Kemudian klik **Administrasi**.
4. Kemudian klik submenu **Kategori Pengumuman**.
5. Sistem akan menampilkan halaman kategori pengumuman. Pada halaman tersebut pengguna dapat:
   - Monitoring daftar kategori pengumuman yang telah ditambahkan.
   - Memperbarui data kategori pengumuman melalui aksi **Edit**.
   - Menghapus data kategori pengumuman melalui aksi **Hapus**.
   - Menambahkan kategori pengumuman melalui tombol **Tambah Kategori**.
