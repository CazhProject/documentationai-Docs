# Anggota — Staff

Admin memiliki kemampuan untuk mengelola data Staff:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Staff**.
3. Sistem akan menampilkan **Halaman Daftar Staff**.
4. Pada halaman daftar Staff, Admin dapat:
   - Menambahkan data staff baru.
   - Import data staff secara massal.
   - Export data staff:
     1. Export seluruh data yang ada pada halaman.
     2. Export berdasarkan data yang dipilih melalui checkbox.
   - Melihat dan mengubah detail data staff.

## Pindah Shift

Admin memiliki kemampuan untuk mengelola data shift Staff:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Staff**.
3. Sistem akan menampilkan **Halaman Daftar Staff**.
4. Perubahan shift staff dapat dilakukan secara massal maupun satuan:
   - **Ubah shift secara massal**
     1. Pilih satu atau beberapa staff menggunakan checkbox, klik **Pindah Shift**.
     2. Sistem akan menampilkan popup untuk memilih shift baru.
   - **Ubah shift secara satuan**
     1. Pilih salah satu staff, gunakan fitur pencarian atau filter untuk menemukan data staff secara spesifik.
     2. Sistem akan menampilkan halaman detail staff yang dipilih.
     3. Pada tab **Data Diri**, Admin dapat mengubah shift staff melalui field **Shift Presensi**.

## Detail Staff

Untuk mengakses detail staff: menu **Anggota** → submenu **Staff** → pilih salah satu staff (gunakan fitur pencarian atau filter). Sistem akan menampilkan halaman detail staff yang dipilih.

### Data Staff

Admin memiliki kemampuan untuk mengelola data staff:

1. Login ke website, lalu klik menu **Anggota**.

4. Pilih salah satu staff, gunakan fitur pencarian atau filter untuk menemukan data staff secara spesifik.

## Cetak Data Diri

Admin memiliki kemampuan untuk mencetak data diri setiap anggota, yaitu Siswa, Guru, dan Staff. Langkah untuk cetak data diri:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Siswa**, **Guru**, atau **Staff**.
3. Sistem akan menampilkan **Halaman Daftar Anggota** yang dipilih.
4. Pilih salah satu anggota, gunakan fitur pencarian atau filter untuk menemukan data anggota secara spesifik.
5. Sistem akan menampilkan halaman detail anggota yang dipilih.
6. Klik tombol **Cetak Data Diri**.
7. Pada halaman tersebut, Admin dapat:
   - Melihat data diri anggota yang dipilih.
   - Menyimpan dokumen data diri anggota.
   - Mencetak data diri langsung melalui printer yang terhubung.
