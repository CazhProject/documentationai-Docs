Menu Kartu digunakan Admin untuk mengelola [Kartu](/referensi/glosarium#kartu) setiap siswa: mengatur limit transaksi dan penarikan harian, mengisi dan menarik saldo kartu, memantau riwayatnya, serta menangani kehilangan kartu. Orang tua dapat melakukan top-up saldo anak melalui aplikasi [CARDS Parent](/cards-parent/top-up-saldo), dan transaksi kartu di kasir dilakukan melalui [CazhPOS](/cazhpos/saldo-dan-kartu).

## List Limit Kartu

Admin dapat mengelola limit kartu setiap siswa dengan mengatur Limit Transaksi Kartu dan Limit Penarikan Kartu.

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Limit Kartu Harian**.
3. Sistem akan menampilkan Data Limit Kartu Harian.

## Atur Limit Transaksi Kartu dan Limit Penarikan Kartu

Admin dapat mengelola limit kartu setiap siswa dengan mengatur Limit Transaksi Harian Kartu dan Limit Penarikan Harian Kartu.

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Limit Kartu Harian**.
3. Sistem akan menampilkan Data Limit Kartu Harian.
4. Pilih salah satu siswa, isi formulir **Limit Transaksi Harian Kartu** dan **Limit Penarikan Harian Kartu**.
5. Klik tombol **Simpan**.

## Riwayat Pergantian Limit Transaksi Kartu

Admin dapat melihat Riwayat Pergantian Limit Transaksi kartu setiap siswa melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Limit Kartu Harian**.
3. Sistem akan menampilkan halaman Limit Kartu Harian.
4. Di halaman ini, Anda dapat klik **Riwayat Limit Transaksi**.
5. Sistem akan menampilkan halaman Riwayat Pergantian Limit Transaksi.

## Riwayat Pergantian Limit Penarikan Kartu

Admin dapat melihat Riwayat Limit Penarikan kartu setiap siswa melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Limit Kartu Harian**.
3. Sistem akan menampilkan halaman Limit Kartu Harian.
4. Di halaman ini, Anda dapat klik **Riwayat Limit Penarikan**.
5. Sistem akan menampilkan halaman Riwayat Pergantian Limit Penarikan.

## Isi Kartu Satuan

Admin dapat mengisi kartu setiap siswa melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Isi Kartu**.
3. Sistem akan menampilkan halaman Isi Kartu.
4. Di halaman ini, Anda dapat mengisi form isi kartu:
   - Sumber Dana
   - Instansi
   - Nomor Kartu
   - Nominal Isi Kartu
5. Klik tombol **Simpan**.

## Isi Kartu Massal

Admin dapat mengisi kartu siswa secara massal melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Isi Kartu**.
3. Sistem akan menampilkan halaman Isi Kartu.
4. Di halaman ini, klik tombol **Massal**.
5. Isi form isi kartu:
   - Sumber Dana
   - Upload Data Excel
6. Klik tombol **Simpan**.

## Riwayat Isi Kartu

Admin dapat melihat Riwayat Isi Kartu setiap siswa melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Riwayat Isi Kartu**.
3. Sistem akan menampilkan halaman Riwayat Isi Kartu.

### Detail Riwayat Isi Kartu

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Riwayat Isi Kartu**.
3. Sistem akan menampilkan halaman Riwayat Isi Kartu.
4. Klik aksi **Detail**.
5. Sistem akan menampilkan Detail Riwayat Isi Kartu.

## Penarikan Saldo Kartu Satuan

Admin dapat menarik saldo kartu setiap siswa melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Kartu**.
2. Klik submenu **Penarikan Saldo**.
3. Sistem akan menampilkan halaman Penarikan Saldo.
4. Di halaman ini, klik tombol **Satuan**.
5. Isi form penarikan kartu:
   - Instansi
   - Nominal
   - Catatan
   - Nomor Kartu
6. Klik tombol **Simpan**.
