Menu Kartu digunakan admin sekolah untuk mengelola [Kartu](/referensi/glosarium#kartu) setiap siswa: mengatur limit transaksi dan penarikan harian, mengisi dan menarik saldo kartu, memantau riwayatnya, serta menangani kehilangan kartu.

<Callout kind="info">
Orang tua dapat melakukan top-up saldo anak melalui aplikasi [CARDS Parent](/cards-parent/kartu), dan transaksi kartu di kasir dilakukan melalui [CazhPOS](/cazhpos/saldo-dan-kartu).
</Callout>

## Melihat List Limit Kartu

Halaman ini menampilkan data limit transaksi dan limit penarikan harian kartu setiap siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Limit Kartu Harian">Klik submenu **Limit Kartu Harian**.</Step>
  <Step title="Lihat data">Sistem menampilkan Data Limit Kartu Harian.</Step>
</Steps>

### Informasi yang ditampilkan

Tabel Limit Kartu Harian menampilkan kolom: **Nama**, **No Kartu**, **Jenis Anggota**, **Instansi**, **Limit Transaksi Harian**, dan **Limit Penarikan**. Data dapat difilter berdasarkan **Instansi**, **Tahun Ajaran**, **Jenis Anggota**, **Tingkat**, **Kelas**, dan **Jurusan** — secara default sistem menampilkan instansi pertama Anda dengan tahun ajaran aktif, dan hanya anggota berstatus aktif.

## Atur Limit Transaksi dan Limit Penarikan Kartu

Atur batas Limit Transaksi Harian Kartu dan Limit Penarikan Harian Kartu per siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Limit Kartu Harian">Klik submenu **Limit Kartu Harian**. Sistem menampilkan Data Limit Kartu Harian.</Step>
  <Step title="Isi limit siswa">Pilih salah satu siswa, lalu isi formulir **Limit Transaksi Harian Kartu** dan **Limit Penarikan Harian Kartu**.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

### Isian formulir

| Isian | Ketentuan |
| --- | --- |
| Limit Transaksi Harian Kartu | Wajib, angka, minimal 0. |
| Limit Penarikan Harian Kartu | Wajib, angka, minimal 0. |

<Callout kind="info">
Limit juga dapat diatur secara massal dengan memilih beberapa siswa melalui checkbox. Nilai limit **0** berarti tanpa batas — sistem tidak memberlakukan pembatasan pada kartu tersebut.
</Callout>

## Riwayat Pergantian Limit Transaksi Kartu

Lihat histori perubahan limit transaksi kartu setiap siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Limit Kartu Harian">Klik submenu **Limit Kartu Harian**. Sistem menampilkan halaman Limit Kartu Harian.</Step>
  <Step title="Buka riwayat">Klik **Riwayat Limit Transaksi**. Sistem menampilkan halaman Riwayat Pergantian Limit Transaksi.</Step>
</Steps>

## Riwayat Pergantian Limit Penarikan Kartu

Lihat histori perubahan limit penarikan kartu setiap siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Limit Kartu Harian">Klik submenu **Limit Kartu Harian**. Sistem menampilkan halaman Limit Kartu Harian.</Step>
  <Step title="Buka riwayat">Klik **Riwayat Limit Penarikan**. Sistem menampilkan halaman Riwayat Pergantian Limit Penarikan.</Step>
</Steps>

## Isi Kartu Satuan

Tambahkan saldo ke kartu satu siswa secara langsung.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Isi Kartu">Klik submenu **Isi Kartu**. Sistem menampilkan halaman Isi Kartu.</Step>
  <Step title="Isi form">Isi form isi kartu: Sumber Dana, Instansi, Nomor Kartu, dan Nominal Isi Kartu.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

### Isian formulir

| Isian | Ketentuan |
| --- | --- |
| Sumber Dana | Wajib. Dipilih dari [Cazhbox](/referensi/glosarium#cazhbox) instansi Anda; saldo Cazhbox ditampilkan pada pilihan. |
| Instansi | Wajib (mode satuan). |
| Nomor Kartu | Wajib (mode satuan), angka, harus milik anggota **aktif** pada instansi tersebut. |
| Nominal Isi Kartu | Wajib (mode satuan), angka, minimal Rp 1. |

<Callout kind="info">
Isi kartu dikenakan biaya admin top-up sesuai pengaturan biaya instansi.
</Callout>

## Isi Kartu Massal

Isi saldo banyak kartu siswa sekaligus lewat upload file Excel.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Isi Kartu">Klik submenu **Isi Kartu**. Sistem menampilkan halaman Isi Kartu.</Step>
  <Step title="Pilih mode massal">Klik tombol **Massal**.</Step>
  <Step title="Isi form">Isi form isi kartu: Sumber Dana dan Upload Data Excel.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

<Callout kind="info">
File upload wajib berformat **.xls** dengan ukuran maksimal **255 KB**. Semua anggota dalam file harus berstatus aktif — jika ada anggota tidak aktif, proses ditolak.
</Callout>

## Riwayat Isi Kartu

Pantau semua transaksi isi kartu yang pernah dilakukan.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Riwayat Isi Kartu">Klik submenu **Riwayat Isi Kartu**. Sistem menampilkan halaman Riwayat Isi Kartu.</Step>
  <Step title="Lihat detail (opsional)">Klik aksi **Detail** pada salah satu data. Sistem menampilkan Detail Riwayat Isi Kartu.</Step>
</Steps>

### Informasi yang ditampilkan

Halaman menampilkan ringkasan **Total Topup** dan **Total Biaya Admin**, dengan filter rentang tanggal (default: awal sampai akhir bulan berjalan). Data dapat diekspor ke Excel dengan nama file **Riwayat Isi Kartu.xlsx**.

## Penarikan Saldo Kartu Satuan

Tarik saldo dari kartu satu siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Penarikan Saldo">Klik submenu **Penarikan Saldo**. Sistem menampilkan halaman Penarikan Saldo.</Step>
  <Step title="Pilih mode satuan">Klik tombol **Satuan**.</Step>
  <Step title="Isi form">Isi form penarikan kartu: Instansi, Nominal, Catatan, dan Nomor Kartu.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

### Isian formulir

| Isian | Ketentuan |
| --- | --- |
| Instansi | Wajib; Anda harus terdaftar pada instansi tersebut. |
| Nomor Kartu | Wajib, angka, anggota harus terdaftar pada instansi tersebut. |
| Wallet Kartu | Wajib; pilihan wallet menampilkan saldo masing-masing. |
| Nominal | Wajib, angka, minimal Rp 1. |
| Catatan | Opsional, maksimal 255 karakter. |

<Callout kind="info">
Aturan sistem: saldo wallet harus mencukupi **nominal + biaya admin penarikan**; proses penarikan melalui dua langkah dan diakhiri verifikasi **PIN**. Jika nominal melebihi sisa Limit Penarikan Harian, transaksi ditolak dan sistem menampilkan sisa limit beserta waktu limit di-reset.
</Callout>

## Penarikan Saldo Kartu Massal

Tarik saldo banyak kartu siswa sekaligus lewat upload file Excel.

<Steps>
  <Step title="Buka Penarikan Saldo">Login ke website, klik menu **Kartu**, lalu klik submenu **Penarikan Saldo**. Sistem menampilkan halaman Penarikan Saldo.</Step>
  <Step title="Pilih mode massal">Klik tombol **Massal**.</Step>
  <Step title="Isi form">Isi form penarikan kartu: Instansi dan Upload Data Excel.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

<Callout kind="info">
File upload wajib berformat **.xls** dengan ukuran maksimal **255 KB**.
</Callout>

### Informasi yang ditampilkan

Halaman menampilkan ringkasan **Total Penarikan** dan **Total Biaya Admin**, dengan filter rentang tanggal (default: bulan berjalan). Data dapat diekspor ke Excel dengan nama file **Riwayat Penarikan Kartu.xlsx**.

## Kehilangan Kartu

Lihat data kehilangan kartu yang dilaporkan setiap siswa.

<Steps>
  <Step title="Buka menu Kartu">Login ke website, lalu klik menu **Kartu**.</Step>
  <Step title="Buka Kehilangan Kartu">Klik submenu **Kehilangan Kartu**.</Step>
  <Step title="Lihat data">Sistem menampilkan halaman Kehilangan Kartu.</Step>
</Steps>

### Informasi yang ditampilkan

Tabel Kehilangan Kartu menampilkan kolom: **Status**, **Tanggal Permohonan**, **Ref ID**, **Operator**, **Nomor Kartu**, **Nama Lengkap**, **Tipe Anggota**, dan **Instansi**. Status permohonan terdiri dari:

| Status | Keterangan |
| --- | --- |
| **Menunggu** | Permohonan menunggu persetujuan. |
| **Disetujui** | Permohonan kehilangan kartu disetujui. |
| **Ditolak** | Permohonan ditolak. |

Data dapat diekspor ke Excel dengan nama file **Laporan Kartu Hilang** (export menyertakan kolom tambahan **Alasan Hilang**).

<Callout kind="info">
Halaman detail hanya dapat dibuka untuk permohonan yang sudah berstatus **Disetujui**.
</Callout>
