# Anggota — Guru

Admin memiliki kemampuan untuk mengelola data Guru, termasuk mengubah status mereka menjadi alumni guru.

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Guru**.
4. Pada halaman Guru, Admin dapat mengelola **Tambah**, **Edit** detail data guru, dan mengubah guru menjadi **Alumni**.

## Ubah Jenis Guru

Admin memiliki kemampuan untuk mengubah jenis guru berdasarkan jenis guru yang dibuat pada pengaturan. Perubahan bisa dilakukan secara satuan maupun massal.

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Guru**.
4. Pada halaman Guru, checklist salah satu guru atau beberapa guru, lalu klik tombol **Ubah Jenis Guru**.
5. Pilih jenis guru yang tersedia, klik tombol **Simpan**.

## Download Kartu Guru

Admin dapat mengunduh [kartu](/referensi/glosarium#kartu) sementara guru dengan langkah berikut:

1. Login ke website.
2. Klik menu **Anggota**.
3. Pilih submenu **Guru**.
4. Sistem akan menampilkan halaman **Guru**.
5. Pilih salah satu data guru dengan mencentang checkbox pada daftar guru.
6. Klik opsi **Unduh Kartu** pada bagian atas daftar data guru.
7. Pilih menu **Unduh** untuk mengunduh kartu guru.
8. Setelah file berhasil diunduh, Admin dapat mencetak kartu sesuai kebutuhan.

## Download Surat Guru

Admin dapat mengunduh surat guru dengan langkah berikut:

1. Login ke website.
2. Klik menu **Anggota**.
3. Pilih submenu **Guru**.
4. Sistem akan menampilkan halaman **Guru**.
5. Pada halaman Guru, pilih data guru dengan mencentang checkbox pada daftar guru.
6. Klik opsi **Download Surat Guru** pada bagian atas daftar guru.
7. Pilih menu **Unduh** untuk mengunduh surat guru.
8. Setelah proses unduh selesai, Admin dapat mencetak surat guru sesuai kebutuhan.

## Ubah Status

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Guru**.
4. Klik checkbox pada nama guru yang ingin diubah statusnya.
5. Pilih **Ubah Status**.
6. Lalu pilih status sesuai kebutuhan.
7. Klik **Simpan** untuk menyimpan perubahan.

## Pindah Instansi

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Guru**.
4. Klik checkbox pada nama guru yang ingin dipindah instansinya.
5. Pilih **Pindah Instansi**.
6. Lalu pilih status dan jenis guru sesuai kebutuhan.
7. Klik **Simpan** untuk menyimpan perubahan.

## Pindah Guru ke Alumni

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Guru**.
4. Klik checkbox pada nama guru yang ingin dipindah ke alumni.
5. Pilih **Pindah Alumni**.
6. Klik **Simpan** untuk menyimpan perubahan.

## Kelola Data Diri Guru

Admin memiliki kemampuan untuk mengelola data diri guru:

1. Login ke website, klik menu **Anggota**.
2. Pilih submenu **Guru**.
3. Klik menu **Edit** pada samping kanan nama guru.
4. Sistem akan menampilkan profil guru.
5. Pilih navbar **Data Guru**, lalu gulir ke bawah.
6. Isi form dengan data yang valid.
7. Pada data diri guru, Admin dapat:
   - Mencetak data diri guru.
   - Melengkapi informasi data diri.
   - Edit informasi data diri.

## Detail Guru

Untuk mengakses detail guru: menu **Anggota** → submenu **Guru** → pilih salah satu guru (gunakan fitur pencarian atau filter untuk menemukan data guru secara spesifik). Sistem akan menampilkan halaman detail guru yang dipilih.

### Kartu

Admin memiliki kemampuan untuk memonitor [kartu](/referensi/glosarium#kartu) guru:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Guru**.
3. Sistem akan menampilkan **Halaman Daftar Guru**.
4. Pilih salah satu guru, gunakan fitur pencarian atau filter untuk menemukan data guru secara spesifik.
5. Sistem akan menampilkan halaman detail guru yang dipilih.
6. Klik tab **Kartu**.
7. Pada halaman Kartu, Admin dapat:
   - Melihat informasi kartu.
   - Melihat akun [CARDS Parents](/cards-parent/pengenalan) dan [CARDS Edu](/cards-edu/pengenalan) yang terhubung.
   - Melihat riwayat transaksi kartu.
   - Melakukan pengaturan kartu (laporkan kartu hilang, reset PIN, blokir, download kartu, dan download surat).

## Jadikan Guru Aktif

Admin memiliki kemampuan untuk menjadikan alumni guru menjadi guru aktif kembali:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **Alumni**.
3. Pilih menu **Guru**.
4. Pilih guru yang akan diaktifkan.
5. Pilih **Ubah ke Guru Aktif** → **Simpan**.
6. Pada halaman jadikan guru aktif, Admin dapat mengubah status guru alumni menjadi guru aktif.

Lihat juga [Anggota — Alumni](/cards-school/anggota-alumni) untuk pengelolaan data alumni secara umum.
