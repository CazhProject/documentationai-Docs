Modul Siswa adalah pusat pengelolaan data siswa bagi admin sekolah — mulai dari menambah dan mengubah data, memindahkan siswa antar kelas atau tahun ajaran, hingga menjadikan siswa sebagai alumni. Seluruh fitur diakses melalui menu **Anggota** → submenu **Siswa**, yang menampilkan **Halaman Siswa**.

Pada halaman Siswa, admin dapat mengelola:

- Tambah data siswa secara satuan dan massal.
- Edit detail data siswa.
- Mengubah siswa menjadi Alumni.
- Export data siswa menjadi file Excel.

## Download Kartu Sementara

Mengunduh dan mencetak [kartu](/referensi/glosarium#kartu) sementara untuk siswa terpilih.

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Siswa**.</Step>
  <Step title="Pilih siswa">Klik checkbox pada nama siswa yang ingin diunduh kartu sementaranya. Sistem akan menampilkan opsi pada bagian atas data siswa.</Step>
  <Step title="Unduh kartu">Pilih opsi **Unduh Kartu**, lalu klik **Unduh**. Sistem akan redirect ke **Card Download**.</Step>
  <Step title="Cetak">Pilih **Cetak** untuk mencetak kartu.</Step>
</Steps>

## Download Surat Siswa

Mengunduh surat berisi panduan penggunaan aplikasi siswa — lihat [CARDS Edu](/cards-edu/tentang).

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Siswa**.</Step>
  <Step title="Pilih siswa">Centang checkbox pada nama siswa yang ingin diunduh suratnya.</Step>
  <Step title="Unduh surat">Klik menu dropdown **Aksi** di bagian atas tabel data, pilih opsi **Download Surat Siswa**, lalu klik tombol **Download** pada pop-up yang muncul.</Step>
  <Step title="Selesai">Sistem akan otomatis mengunduh dan menyimpan file ke perangkat.</Step>
</Steps>

## Pindah Data Siswa

Opsi **Pindah Data Siswa** memuat sejumlah aksi massal untuk memindahkan atau memperbarui data siswa terpilih. Alur dasarnya sama untuk semua aksi.

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Siswa**.</Step>
  <Step title="Pilih siswa">Klik checkbox pada siswa yang datanya ingin dipindah atau diubah. Sistem akan menampilkan opsi pada bagian atas data siswa.</Step>
  <Step title="Pilih aksi">Pilih opsi **Pindah Data Siswa**, lalu pilih salah satu aksi sesuai tabel di bawah.</Step>
  <Step title="Isi data dan simpan">Isi data yang diminta pada formulir yang muncul, lalu klik **Simpan** untuk menyimpan perubahan.</Step>
</Steps>

| Aksi | Data yang diisi |
| --- | --- |
| **Pindah Tahun Ajaran** | Data tahun ajaran tujuan. |
| **Pindah Tingkat** | Data tingkat tujuan. |
| **Pindah Kelas** | Data kelas tujuan. |
| **Pindah Jurusan** | Data jurusan tujuan. |
| **Pindah Alumni** | Tidak ada isian tambahan — langsung klik **Simpan**. |
| **Pindah Instansi** | Data instansi tujuan. |
| **Ubah Status** | Isi atau pilih data status baru. |
| **Tambah Tag Siswa** | Pilih data tag yang ingin ditambahkan pada siswa. |

<Callout kind="info">Siswa yang dipindah ke alumni akan berpindah pengelolaannya ke menu [Anggota — Alumni](/cards-school/anggota-alumni) dan dapat diaktifkan kembali dari sana.</Callout>

## Tambah dan Import Siswa

Menambahkan siswa baru secara satuan atau massal.

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Siswa** yang telah terdaftar.</Step>
  <Step title="Tambah atau import">**Tambah Siswa**: klik tombol **Tambah Siswa** → isi **Data Diri & Orang Tua** → **Simpan**. **Import Siswa**: klik menu **Import Siswa** → isi **Data Siswa** → **Simpan**.</Step>
  <Step title="Selesai">Sistem akan menampilkan pesan notifikasi siswa berhasil ditambahkan.</Step>
</Steps>

## Edit Data Diri Siswa

Memperbarui data diri dan informasi tambahan siswa.

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Daftar Siswa**.</Step>
  <Step title="Buka detail siswa">Pilih **Detail** pada tabel data siswa. Sistem akan menampilkan form **Data Diri Siswa**.</Step>
  <Step title="Edit dan simpan">Isi form, lalu klik **Simpan**.</Step>
</Steps>

Edit data diri dapat digunakan untuk:

- Edit data siswa.
- Edit informasi tambahan siswa.

## Edit Data Orang Tua Siswa

Memperbarui data orang tua siswa, termasuk data penghasilan.

<Steps>
  <Step title="Buka halaman Siswa">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**. Sistem akan menampilkan **Halaman Siswa**.</Step>

## Detail Siswa

Setiap siswa memiliki halaman detail dengan sejumlah tab. Untuk mengaksesnya: menu **Anggota** → submenu **Siswa** → pilih **Detail** pada data siswa yang diinginkan, lalu pilih tab yang diperlukan pada navbar detail profil siswa.

### Kartu Siswa

Buka tab **Kartu** pada detail siswa. Tab ini dapat digunakan untuk:

- Melihat riwayat transaksi.
- Mendapatkan informasi [kartu](/referensi/glosarium#kartu).

### Tagihan Siswa

Buka tab **Tagihan** pada detail siswa untuk mencari [tagihan](/referensi/glosarium#tagihan) siswa. Pada halaman Tagihan terdapat:

- List tagihan.
- Pencarian jenis tagihan.

<Callout kind="info">Tagihan ini juga tampil di aplikasi orang tua — lihat [Bayar Tagihan di CARDS Parents](/cards-parent/keuangan).</Callout>

### Dokumen Siswa

Buka tab **Dokumen** pada detail siswa. Admin dapat:

- Menambah dokumen baru.
- Edit dokumen.
- Hapus dokumen.
- Mencari dokumen.

## Cetak Data Diri

Mencetak data diri setiap anggota — berlaku untuk Siswa, Guru, dan Staff.

<Steps>
  <Step title="Buka daftar anggota">Login ke website, klik menu **Anggota**, lalu klik submenu **Siswa**, **Guru**, atau **Staff**. Sistem akan menampilkan **Halaman Daftar Anggota** yang dipilih.</Step>
  <Step title="Pilih anggota">Pilih salah satu anggota; gunakan fitur pencarian atau filter untuk menemukan data anggota secara spesifik. Sistem akan menampilkan halaman detail anggota yang dipilih.</Step>
  <Step title="Cetak">Klik tombol **Cetak Data Diri**.</Step>
</Steps>

Pada halaman tersebut, Admin dapat:

- Melihat data diri anggota yang dipilih.
- Menyimpan dokumen data diri anggota.
- Mencetak data diri langsung melalui printer yang terhubung.
