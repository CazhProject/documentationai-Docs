# Anggota — Siswa

Admin dapat mengelola data siswa seperti menambah, mengubah, dan menjadikan siswa sebagai alumni.

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Siswa**.
3. Sistem akan menampilkan **Halaman Siswa**.
4. Pada halaman Siswa, Admin dapat mengelola:
   - Tambah data siswa secara satuan dan massal.
   - Edit detail data siswa.
   - Mengubah siswa menjadi Alumni.
   - Export data siswa menjadi file Excel.

## Download Kartu Sementara

Admin dapat mengunduh [kartu](/referensi/glosarium#kartu) sementara siswa:

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Siswa**.
3. Sistem akan menampilkan **Halaman Siswa**.
4. Klik checkbox pada nama siswa yang ingin diunduh kartu sementaranya.
5. Sistem akan menampilkan opsi pada bagian atas data siswa.
6. Pilih opsi **Unduh Kartu**, lalu klik **Unduh**.
7. Sistem akan redirect ke **Card Download**.
8. Pilih **Cetak** untuk mencetak kartu.

## Download Surat Orang Tua

Surat ini berisi panduan penggunaan aplikasi orang tua — lihat [CARDS Parents](/cards-parent/pengenalan).

## Download Surat Siswa

Admin dapat mengunduh surat siswa dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Anggota**.
2. Klik submenu **Siswa**.
3. Sistem akan menampilkan **Halaman Siswa**.
4. Centang checkbox pada nama siswa yang ingin diunduh suratnya.
5. Klik menu dropdown **Aksi** di bagian atas tabel data.
6. Pilih opsi **Download Surat Siswa**.
7. Klik tombol **Download** pada pop-up yang muncul.
8. Sistem akan otomatis mengunduh dan menyimpan file ke perangkat.

Surat ini berisi panduan penggunaan aplikasi siswa — lihat [CARDS Edu](/cards-edu/pengenalan).

## Tambah dan Import Siswa

Admin memiliki kemampuan untuk menambah dan meng-import siswa:

1. Login ke website, kemudian klik menu **Anggota**.
2. Kemudian klik submenu **Siswa**.
3. Sistem akan menampilkan **Halaman Siswa** yang telah terdaftar.
4. **Tambah Siswa**: klik tombol **Tambah Siswa** → isi **Data Diri & Orang Tua** → **Simpan**.
5. **Import Siswa**: klik menu **Import Siswa** → isi **Data Siswa** → **Simpan**.
6. Sistem akan menampilkan pesan notifikasi siswa berhasil ditambahkan.

## Cetak Data Diri

Admin memiliki kemampuan untuk mencetak data diri setiap anggota, yaitu Siswa, Guru, dan Staff. Langkah untuk cetak data diri:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Siswa**, **Guru**, atau **Staff**.
3. Sistem akan menampilkan **Halaman Daftar Anggota** yang dipilih.
4. Pilih salah satu anggota, gunakan fitur pencarian atau filter untuk menemukan data anggota secara spesifik.
5. Sistem akan menampilkan halaman detail anggota yang dipilih.
6. Klik tombol **Cetak Data Diri**.
7. Pada halaman tersebut, Admin dapat:
   - Melihat data diri anggota yang dipilih.
   - Menyimpan dokumen data diri anggota.
   - Mencetak data diri langsung melalui printer yang terhubung.
