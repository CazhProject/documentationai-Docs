Menu Tabungan digunakan Admin untuk mengelola produk tabungan sekolah, rekening tabungan siswa, serta transaksi setoran dan penarikan — baik satuan maupun massal. Setoran dan penarikan tunai di kasir dapat dilakukan melalui [CazhPOS](/cazhpos/saldo-dan-kartu).

## List Produk Tabungan

Admin dapat mengelola Produk Tabungan: menambah produk, mengedit produk, dan menghapus produk tabungan.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Produk**.
4. Halaman ini menampilkan list Produk Tabungan yang dibuat oleh admin sekolah.
5. Admin dapat mengatur Produk Tabungan sesuai kebutuhan sekolah.

## Tambah Produk Tabungan

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Produk**, klik tombol **Tambah Produk**.
4. Halaman ini menampilkan formulir yang harus diisi untuk menambah Produk Tabungan.
5. Setelah Produk Tabungan ditambahkan, produk akan muncul pada list Produk Tabungan.

## Edit Produk Tabungan

Admin dapat mengedit Produk Tabungan: mengganti instansi, mengubah biaya penarikan, mengubah status produk tabungan, dan lainnya.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Produk**, pilih salah satu produk, klik aksi **Edit**.
4. Halaman Edit Produk Tabungan memungkinkan Admin untuk mengubah produk tabungan yang sudah ada sebelumnya.

## List Rekening Tabungan

Admin dapat mengelola Rekening Tabungan: menambah rekening, mengedit rekening, melihat detail rekening, mencari, mengekspor, dan menghapus Rekening Tabungan.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Rekening**.
4. Halaman Daftar Rekening menampilkan rekening yang dimiliki oleh para siswa.

## Detail Rekening Tabungan

Admin dapat mengelola Detail Rekening Tabungan per siswa: membuat transaksi tabungan, menonaktifkan rekening, mencetak transaksi, memblokir rekening, dan mengekspor data transaksi rekening.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Rekening**.
4. Pilih salah satu rekening, klik **Detail Rekening**.

## Tambah Rekening Tabungan Satuan

Admin dapat menambah rekening baru untuk masing-masing siswa, sesuai dengan instansi dan Produk Tabungan yang ada pada instansi tersebut.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Rekening**.
4. Klik tombol **Tambah Rekening**.
5. Isi formulir Tambah Rekening Satuan, klik **Simpan**.

## Tambah Rekening Tabungan Massal

Admin dapat menambah rekening baru secara massal untuk siswa dalam satu instansi dan tipe anggota yang sama.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Rekening**.
4. Klik tombol **Tambah Rekening**.
5. Isi formulir Tambah Rekening Massal, klik **Simpan**.

## Transaksi dan Riwayat Transaksi Tabungan

Halaman ini menampilkan Riwayat Transaksi Tabungan dari seluruh siswa dan seluruh Produk Tabungan. Admin dapat melihat total setoran tunai, setoran non-tunai, dan penarikan tunai. Admin juga dapat mencari, memfilter, dan mengekspor data Transaksi Tabungan.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Transaksi**.

## Setoran Massal

Fitur yang digunakan Admin untuk melakukan transaksi setor tabungan secara massal. Setoran massal dibuat dengan cara upload data Excel setoran yang diambil dari template sesuai filter yang diatur.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Transaksi**.
4. Klik tombol **Transaksi Masal**.
5. Isi formulir dan unduh template.
6. Isi template dengan data yang diinginkan.
7. Unggah template kembali pada formulir Transaksi Masal.

## Penarikan Massal

Fitur yang digunakan Admin untuk melakukan transaksi tarik tabungan secara massal. Penarikan massal dibuat dengan cara upload data Excel penarikan yang diambil dari template sesuai filter yang diatur.

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Tabungan**.
3. Klik submenu **Transaksi**.
4. Klik tombol **Transaksi Masal**, pilih tab **Penarikan**.
5. Isi formulir dan unduh template.
6. Isi template dengan data yang diinginkan.
7. Unggah template kembali pada formulir Transaksi Masal.
