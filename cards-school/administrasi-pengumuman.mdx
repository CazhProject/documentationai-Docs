---

Menu Pengumuman digunakan Admin untuk menyebarkan informasi ke platform Cards Edu, Cards Parents, maupun Landing Page Sekolah.

## List Pengumuman

Admin memiliki kemampuan untuk mengelola daftar pengumuman dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Pengumuman**.
4. Klik menu **List Pengumuman**.
5. Sistem akan menampilkan halaman daftar pengumuman.
6. Pada halaman tersebut, Admin dapat:
   - Melihat daftar data pengumuman.
   - Mencari data pengumuman.

## Edit Pengumuman

Admin memiliki kemampuan untuk mengubah data pengumuman dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Pengumuman**.
4. Klik menu **List Pengumuman**.
5. Sistem akan menampilkan halaman daftar pengumuman.
6. Cari data pengumuman yang ingin diubah.
7. Klik ikon **Titik Tiga** pada data pengumuman yang dipilih.
8. Pilih menu **Edit**.
9. Ubah informasi pengumuman sesuai kebutuhan.
10. Klik tombol **Simpan** untuk menyimpan perubahan data pengumuman.

## Arsipkan Pengumuman

Admin memiliki kemampuan untuk mengarsipkan data pengumuman dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Pengumuman**.
4. Klik menu **List Pengumuman**.
5. Sistem akan menampilkan halaman daftar pengumuman.
6. Cari data pengumuman yang ingin diarsipkan.
7. Klik ikon **Titik Tiga** pada data pengumuman yang dipilih.
8. Pilih menu **Arsipkan**.
9. Sistem akan memindahkan data pengumuman ke daftar arsip pengumuman.

## Hapus Pengumuman

Admin memiliki kemampuan untuk menghapus data pengumuman dengan langkah berikut:

1. Login ke website.
2. Masuk ke menu **Administrasi**.
3. Pilih submenu **Pengumuman**.
4. Klik menu **List Pengumuman**.
5. Sistem akan menampilkan daftar pengumuman.
6. Temukan data pengumuman yang akan dihapus.
7. Klik ikon **Titik Tiga** pada baris data pengumuman.
8. Pilih opsi **Hapus Pengumuman**.
9. Sistem akan menghapus data pengumuman dari daftar pengumuman.

## Tambah Pengumuman

Admin dapat menambahkan data pengumuman baru melalui langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Pengumuman**.
4. Klik menu **List Pengumuman**.
5. Sistem akan menampilkan halaman daftar pengumuman.
6. Klik tombol **Tambah Pengumuman**.
7. Isi formulir pengumuman sesuai kebutuhan.
8. Pada bagian Platform, Admin dapat memilih satu atau lebih platform tujuan pengumuman, meliputi:
   - Cards Edu
   - Cards Parents
   - Landing Page Sekolah
9. Pengumuman akan ditampilkan sesuai platform yang dipilih.
10. Klik tombol **Simpan** untuk menyimpan data pengumuman.
