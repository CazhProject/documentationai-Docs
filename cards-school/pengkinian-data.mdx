# Pengkinian Data

Fitur Pengkinian Data memungkinkan lembaga membuka jalur pengajuan perubahan atau pembaruan data oleh pengguna (misalnya orang tua melalui aplikasi [CARDS Parents](/cards-parent/anggota/pengkinian-data)). Admin mengelola pengaturannya melalui menu **Anggota** → submenu **Pengkinian Data**.

## List Pengaturan Pengkinian Data

List Pengkinian Data memungkinkan Admin melihat dan mengelola pengkinian data yang telah dibuat. Langkah-langkahnya sebagai berikut:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Pengkinian Data**.
3. Sistem akan menampilkan list data pengaturan pengkinian data yang telah dibuat.
4. Pada halaman Pengkinian Data, Admin dapat melakukan **Search** data pengkinian data yang diinginkan.

## Tambah Pengaturan Pengkinian Data

Tambah Pengkinian Data memungkinkan Admin menambahkan pengaturan pengkinian data yang dapat digunakan pengguna untuk mengajukan perubahan atau pembaruan data. Langkah-langkahnya sebagai berikut:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Pengkinian Data**.
3. Sistem akan menampilkan list data pengaturan pengkinian data yang telah dibuat.
4. Klik tombol **Tambah Pengaturan**.
5. Input data, lalu klik tombol **Simpan**.

## Edit Pengaturan Pengkinian Data

Edit Pengkinian Data memungkinkan Admin mengubah pengaturan pengkinian data yang telah dibuat sesuai kebutuhan. Langkah-langkahnya sebagai berikut:

1. Login ke website, lalu klik menu **Anggota**.
2. Kemudian, klik submenu **Pengkinian Data**.
3. Sistem akan menampilkan list data pengaturan pengkinian data yang telah dibuat.
4. Klik tombol aksi **Edit**.
5. Ubah data yang sudah ada, lalu klik tombol **Simpan**.
