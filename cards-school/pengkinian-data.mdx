title: Pengkinian Data
description: >-
  Panduan pengaturan pengkinian data di CARDS School: melihat list, menambah,
  dan mengedit pengaturan pengkinian data untuk pengajuan pembaruan data oleh
  pengguna.
---
Fitur Pengkinian Data memungkinkan admin sekolah membuka jalur pengajuan perubahan atau pembaruan data oleh pengguna — misalnya orang tua melalui aplikasi [CARDS Parents](/cards-parent/akun). Pengaturannya dikelola melalui menu **Anggota** → submenu **Pengkinian Data**.

Setiap pengaturan menghasilkan tautan publik berformat `{alamat-website}/pd/{link}` yang dapat dibagikan; anggota membuka tautan tersebut untuk memperbarui datanya sendiri selama periode yang ditentukan.

## List Pengaturan

Link Vidio: [https://drive.google.com/file/d/1dt9kHsNdhSnrUfgGiQSN5HZbMfImapwS/view?usp=drive\_link](https://drive.google.com/file/d/1dt9kHsNdhSnrUfgGiQSN5HZbMfImapwS/view?usp=drive_link)

**List Pengkinian Data** memungkinkan admin melihat dan mengelola pengkinian data yang telah dibuat. Langkah-langkah sebagai berikut :

1. Login ke website, lalu klik menu **Anggota**.

2. Kemudian, klik submenu **Pengkinian Data**.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784823917808-6m7t35y5r48-pasted-image-1784823915670.png?fm=auto&auto=compress%2Cformat" width="1892" height="1000" alt="" />
3. Sistem akan menampilkan List data pengaturan pengkinian data yang telah dibuat.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784824230631-oxflsnuloq-pasted-image-1784824228454.png?fm=auto&auto=compress%2Cformat" width="1878" height="990" alt="" />
4. Pada Halaman Pengkinian Data, Admin dapat Search Data Pengkinian Data yang diinginkan.

## Tambah Pengaturan

Link Vidio: [https://drive.google.com/file/d/1Y2fxbVu37O\_YXUImXC8fqToyKZNl-9s8/view?usp=drive\_link](https://drive.google.com/file/d/1Y2fxbVu37O_YXUImXC8fqToyKZNl-9s8/view?usp=drive_link)

1. Login ke website, lalu klik menu **Anggota**.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784823884629-zrz4qu7deto-pasted-image-1784823882540.png?fm=auto&auto=compress%2Cformat" width="1872" height="996" alt="" />
2. Kemudian, klik submenu **Pengkinian Data**.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784823917808-6m7t35y5r48-pasted-image-1784823915670.png?fm=auto&auto=compress%2Cformat" width="1892" height="1000" alt="" />
3. Sistem akan menampilkan List data pengaturan pengkinian data yang telah dibuat.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784824261848-25nwutrdvz9-pasted-image-1784824237924.png?fm=auto&auto=compress%2Cformat" width="1878" height="990" alt="" />
4. Klik button Tambah Pengaturan.
   <Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784824289370-ebrntmbrc49-pasted-image-1784824287594.png?fm=auto&auto=compress%2Cformat" width="1890" height="990" alt="" />
