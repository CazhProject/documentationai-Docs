Menu Rapor pada modul Akademik digunakan admin untuk mengelola grup rapor siswa. Grup rapor berisi rapor siswa yang dikelompokkan berdasarkan kelas, [tahun ajaran](/referensi/glosarium#tahun-ajaran), dan [semester](/referensi/glosarium#semester).

## Melihat List Grup Rapor Siswa

Admin dapat melihat list grup rapor siswa yang berisi rapor siswa berdasarkan kelas siswa.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Sistem akan menampilkan **Grup Rapor**.

## Detail Grup Rapor Siswa

Admin dapat melihat detail grup rapor siswa untuk mengetahui siswa-siswa yang tergabung di dalamnya.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Sistem akan menampilkan **Grup Rapor**.
4. Klik aksi **Detail**.
5. Sistem akan menampilkan list siswa-siswa yang tergabung dalam grup rapor tersebut.

## Edit Grup Rapor Siswa

Admin dapat mengedit grup rapor siswa berdasarkan tahun ajaran dan semester.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Sistem akan menampilkan **Grup Rapor**.
4. Klik aksi **Edit**.
5. Sistem akan menampilkan formulir.
6. Input formulir, lalu klik tombol **Simpan Perubahan**.

## Hapus Grup Rapor Siswa

Admin dapat menghapus grup rapor siswa yang sudah tidak digunakan.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Sistem akan menampilkan **Grup Rapor**.
4. Klik aksi **Hapus**.
5. Sistem akan menampilkan popup konfirmasi.
6. Klik tombol **Hapus**.

## Tambah Grup Rapor Siswa

Admin dapat menambah grup rapor siswa berdasarkan data yang diperlukan.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Sistem akan menampilkan **Grup Rapor**.
4. Klik tombol **Tambah Grup Rapor**.
5. Isi formulir **Tambah Grup Rapor**.
6. Klik tombol **Simpan**.

## Hapus Siswa dari Grup Rapor

Fitur ini digunakan untuk menghapus nama siswa dari sebuah grup rapor.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Rapor**.
3. Klik **Aksi** pada grup rapor siswa yang ingin dikelola.
4. Lalu klik **Detail**.
5. Sistem akan menampilkan **Detail Rapor Siswa**.
6. Pilih opsi **Aksi** pada nama siswa yang ingin dihapus.
7. Lalu klik **Hapus**.
