Menu Donasi digunakan admin sekolah untuk mengelola program donasi instansi — membuat, mengubah, menghapus, dan memantau donatur. Terdapat tiga jenis program donasi:

| Jenis | Keterangan | Tab |
| --- | --- | --- |
| **Donasi Umum** | Donasi tanpa target dana. | **Donasi Umum** |
| **Donasi Penggalangan Dana** | Donasi dengan target dana. | **Penggalangan Dana** |
| **Donasi Khusus** | Donasi berbasis kuota dan nominal. | **Donasi Khusus** |

## Daftar Donasi

Melihat dan mengelola daftar program donasi per jenis.

<Steps>
  <Step title="Buka Donasi">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**.</Step>
  <Step title="Pilih tab program">Klik tab sesuai program donasi (lihat tabel di atas). Sistem akan menampilkan halaman Daftar Donasi sesuai dengan program donasi yang dipilih.</Step>
  <Step title="Kelola daftar">Tampilkan daftar donasi secara spesifik menggunakan pencarian, export daftar donasi, tambah program donasi, perbarui program donasi, hapus program donasi, dan tampilkan setiap detail donasi.</Step>
</Steps>

## Tambah Donasi

Membuat program donasi baru.

<Steps>
  <Step title="Buka Donasi">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**. Sistem akan menampilkan halaman Daftar Donasi tab Umum.</Step>
  <Step title="Buka formulir">Klik tombol **Tambah Program Donasi**. Sistem akan menampilkan formulir tambah donasi.</Step>
  <Step title="Isi formulir">Pilih jangkauan donasi (internal instansi, eksternal, maupun keduanya), pilih program donasi (Umum, Penggalangan Dana, atau Khusus — lihat tabel di atas), lalu lengkapi seluruh formulir donasi.</Step>
  <Step title="Simpan">Klik **Simpan**; sistem akan menampilkan donasi sesuai dengan jenis program, periode, dan jangkauan donasi.</Step>
</Steps>

## Edit Donasi

Mengubah data program donasi.

<Steps>
  <Step title="Buka Donasi">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**. Sistem akan menampilkan halaman Daftar Donasi tab Umum.</Step>
  <Step title="Pilih donasi">Pilih tab program donasi yang akan diedit, lalu pilih salah satu data donasi; gunakan fitur pencarian untuk menemukan donasi secara spesifik.</Step>
  <Step title="Buka form edit">Klik ikon **Edit** pada baris data donasi yang dipilih. Sistem akan menampilkan halaman Edit Donasi untuk mengubah atau memperbarui data donasi.</Step>
  <Step title="Simpan">Klik **Simpan**; sistem akan menyimpan perubahan.</Step>
</Steps>

## Hapus Donasi

Menghapus program donasi.

<Steps>
  <Step title="Buka Donasi">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**. Sistem akan menampilkan halaman Daftar Donasi tab Umum.</Step>
  <Step title="Pilih donasi">Pilih tab program donasi yang akan dihapus, lalu pilih salah satu data donasi; gunakan fitur pencarian untuk menemukan donasi secara spesifik.</Step>
  <Step title="Hapus">Klik ikon **Hapus** pada baris data donasi yang dipilih. Sistem akan menampilkan popup Konfirmasi Hapus.</Step>
  <Step title="Konfirmasi">Klik **Hapus**; sistem akan menghapus data donasi.</Step>
</Steps>

<Callout kind="alert">Program donasi yang dihapus tidak dapat dikembalikan. Pastikan data yang dipilih sudah benar sebelum mengonfirmasi.</Callout>

## Detail Donasi Umum

Memonitor data donatur pada donasi umum.

<Steps>
  <Step title="Buka tab Umum">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**. Sistem akan menampilkan halaman Daftar Donasi tab **Umum**.</Step>
  <Step title="Pilih donasi">Pilih salah satu data donasi umum; gunakan fitur pencarian untuk menemukan donasi secara spesifik. Klik ikon **Detail** pada baris data donasi yang dipilih.</Step>
  <Step title="Kelola detail">Sistem akan menampilkan halaman detail donasi umum. Admin dapat melihat informasi donasi, melihat total dana yang telah terkumpul, melihat daftar donatur, export daftar donatur, serta melihat dan mencetak bukti pembayaran dari setiap donatur.</Step>
</Steps>

## Detail Donasi Penggalangan Dana

Memonitor data donatur pada donasi penggalangan dana.

<Steps>
  <Step title="Buka tab Penggalangan Dana">Login ke website, klik menu **Keuangan**, lalu klik submenu **Donasi**. Sistem akan menampilkan halaman Daftar Donasi Umum; klik tab **Penggalangan Dana**.</Step>
  <Step title="Pilih donasi">Pilih salah satu data donasi; gunakan fitur pencarian untuk menemukan donasi secara spesifik. Klik ikon **Detail** pada baris data donasi yang dipilih.</Step>
  <Step title="Kelola detail">Sistem akan menampilkan halaman detail donasi penggalangan dana. Admin dapat melihat informasi donasi, melihat target dan total dana yang telah terkumpul, melihat daftar donatur, export daftar donatur, serta melihat dan mencetak bukti pembayaran dari setiap donatur.</Step>
</Steps>
