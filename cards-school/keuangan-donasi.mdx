Menu Donasi digunakan Admin untuk mengelola tiga jenis program donasi: **Donasi Umum** (tanpa target dana), **Donasi Penggalangan Dana** (dengan target dana), dan **Donasi Khusus** (berbasis kuota dan nominal).

## Daftar Donasi

Admin memiliki kemampuan untuk mengelola program donasi. Langkah untuk akses data donasi:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Donasi**.
3. Daftar donasi ditampilkan berdasarkan program donasi:
   - Donasi Umum → klik tab **Donasi Umum**.
   - Donasi Penggalangan Dana → klik tab **Penggalangan Dana**.
   - Donasi Khusus → klik tab **Donasi Khusus**.
4. Sistem akan menampilkan halaman Daftar Donasi sesuai dengan program donasi yang dipilih.
5. Pada daftar donasi, Admin dapat:
   - Menampilkan daftar donasi secara spesifik menggunakan pencarian.
   - Melakukan export daftar donasi.
   - Menambahkan program donasi.
   - Memperbarui program donasi.
   - Menghapus program donasi.
   - Menampilkan setiap detail donasi.

## Tambah Donasi

Admin memiliki kemampuan untuk membuat donasi. Langkah untuk membuat donasi:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Donasi**.
3. Sistem akan menampilkan halaman Daftar Donasi tab Umum.
4. Klik tombol **Tambah Program Donasi**.
5. Sistem akan menampilkan formulir tambah donasi:
   - Pilih jangkauan donasi → donasi dapat dibuat untuk internal instansi, eksternal, maupun keduanya.
   - Pilih program donasi:
     1. **Donasi Umum** → donasi tanpa target dana.
     2. **Donasi Penggalangan Dana** → donasi dengan target dana.
     3. **Donasi Khusus** → donasi berbasis kuota dan nominal.
   - Lengkapi seluruh formulir donasi.
6. Klik **Simpan**, sistem akan menampilkan donasi sesuai dengan jenis program, periode, dan jangkauan donasi.

## Edit Donasi

Admin memiliki kemampuan untuk mengubah data donasi. Langkah untuk mengubah data donasi:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Donasi**.
3. Sistem akan menampilkan halaman Daftar Donasi tab Umum.
4. Pilih tab program donasi yang akan diedit.
5. Sistem akan menampilkan daftar donasi yang dipilih.
6. Pilih salah satu data donasi; gunakan fitur pencarian untuk menemukan donasi secara spesifik.
7. Klik ikon **Edit** pada baris data donasi yang dipilih.
8. Sistem akan menampilkan halaman Edit Donasi; pengguna dapat mengubah atau memperbarui data donasi.
9. Klik **Simpan**, sistem akan menyimpan perubahan.

3. Sistem akan menampilkan halaman Daftar Donasi tab Umum.
4. Pilih tab program donasi yang akan dihapus.
5. Sistem akan menampilkan daftar donasi yang dipilih.

7. Klik ikon **Hapus** pada baris data donasi yang dipilih.
8. Sistem akan menampilkan popup Konfirmasi Hapus.
9. Klik **Hapus**, sistem akan menghapus data donasi.

7. Pada halaman detail donasi, Admin dapat:
