Menu Tahfidz pada modul Kesiswaan digunakan admin sekolah untuk mengelola program [tahfidz](/referensi/glosarium#tahfidz) siswa beserta riwayat perkembangannya.

<Callout kind="info">
Progress tahfidz siswa juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).
</Callout>

## Kelola Tahfidz

Halaman daftar tahfidz menjadi pusat pengelolaan program tahfidz siswa.

<Steps>
  <Step title="Buka menu Kesiswaan">Login ke website, kemudian klik menu **Kesiswaan**.</Step>
  <Step title="Buka Tahfidz">Klik submenu **Tahfidz**. Sistem menampilkan halaman daftar tahfidz.</Step>
  <Step title="Kelola program">Dari halaman ini admin dapat:\n    - Klik **Tambah** untuk membuat program tahfidz baru.\n    - Klik **Edit** untuk mengubah program tahfidz yang ada.\n    - Klik **Detail** untuk menambah riwayat perkembangan tahfidz setiap siswa.\n    - Klik **Hapus** jika ingin menghapus program tahfidz.</Step>
</Steps>

## Detail Tahfidz

Lihat detail data tahfidz beserta anggota dan riwayat progressnya.

<Steps>
  <Step title="Buka Tahfidz">Login ke website, klik menu **Kesiswaan**, lalu klik submenu **Tahfidz**. Sistem menampilkan halaman Tahfidz.</Step>
  <Step title="Buka detail">Klik **Detail**; sistem menampilkan halaman detail tahfidz.</Step>
  <Step title="Kelola detail">Pada halaman ini admin dapat melihat data anggota tahfidz, menambah anggota, dan menambah riwayat progress tahfidz masing-masing anggota.</Step>
</Steps>

## Tambah Tahfidz

Tambahkan data program tahfidz baru.

<Steps>
  <Step title="Buka Tahfidz">Login ke website, klik menu **Kesiswaan**, lalu klik submenu **Tahfidz**. Sistem menampilkan data Tahfidz.</Step>
  <Step title="Buka form tambah">Klik tombol **Tambah Tahfidz**. Sistem menampilkan form tambah.</Step>
  <Step title="Isi data">Pada form tambah, masukkan data dengan benar dan sesuai.</Step>
  <Step title="Simpan">Klik tombol **Simpan**.</Step>
</Steps>

## Edit Tahfidz

Ubah data tahfidz yang sudah ada.

<Steps>
  <Step title="Buka Tahfidz">Login ke website, klik menu **Kesiswaan**, lalu klik submenu **Tahfidz**. Sistem menampilkan halaman Tahfidz.</Step>
  <Step title="Buka form edit">Klik ikon **Edit**.</Step>
  <Step title="Ubah dan simpan">Edit form dengan sesuai dan benar, lalu klik **Simpan**.</Step>
</Steps>

## Export Tahfidz

Unduh data tahfidz dalam format file Excel.

<Steps>
  <Step title="Buka Tahfidz">Login ke website, klik menu **Kesiswaan**, lalu klik submenu **Tahfidz**. Sistem menampilkan halaman Tahfidz.</Step>
  <Step title="Export">Klik ikon **Excel**.</Step>
  <Step title="Buka file">Data tahfidz tersimpan pada perangkat pengguna dalam file Excel.</Step>
</Steps>
