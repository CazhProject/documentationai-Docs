Menu Tahfidz pada modul Kesiswaan digunakan admin untuk mengelola program [tahfidz](/referensi/glosarium#tahfidz) siswa beserta riwayat perkembangannya. Progress tahfidz siswa juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## Kelola Tahfidz

Admin dapat mengelola data tahfidz siswa. Berikut adalah langkah-langkahnya:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Klik submenu **Tahfidz**.
3. Sistem akan menampilkan halaman daftar tahfidz.
4. Klik **Tambah** untuk membuat program tahfidz baru.
5. Klik **Edit** untuk mengubah program tahfidz yang ada.
6. Klik **Detail** untuk menambah riwayat perkembangan tahfidz setiap siswa.
7. Klik **Hapus** jika ingin menghapus program tahfidz.

## Detail Tahfidz

Detail Tahfidz digunakan admin untuk melihat detail data tahfidz, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Tahfidz**.
3. Sistem akan menampilkan halaman Tahfidz.
4. Klik **Detail**; sistem akan menampilkan halaman detail tahfidz.
5. Pada halaman ini admin dapat melihat data anggota tahfidz, menambah anggota, dan menambah riwayat progress tahfidz masing-masing anggota.

## Tambah Tahfidz

Tambah Tahfidz digunakan admin untuk menambahkan data tahfidz baru, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Tahfidz**.
3. Sistem menampilkan data Tahfidz.
4. Klik tombol **Tambah Tahfidz**.
5. Sistem menampilkan form tambah.
6. Pada form tambah, masukkan data dengan benar dan sesuai.
7. Klik tombol **Simpan**.

## Edit Tahfidz

Edit Tahfidz digunakan admin untuk mengedit data tahfidz, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Tahfidz**.
3. Sistem akan menampilkan halaman Tahfidz.
4. Klik ikon **Edit**.
5. Edit form dengan sesuai dan benar, lalu klik **Simpan**.

## Export Tahfidz

Export Tahfidz digunakan admin untuk mengunduh data tahfidz dalam format file Excel, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Tahfidz**.
3. Sistem akan menampilkan halaman Tahfidz.
4. Klik ikon **Excel**.
5. Data tahfidz akan tersimpan pada perangkat pengguna dalam file Excel.
