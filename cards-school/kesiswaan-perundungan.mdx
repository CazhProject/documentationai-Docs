## Tambah Riwayat Perundungan

Link video:&#x20;

Tambahkan peristiwa perundungan terbaru ke dalam sebuah laporan. Riwayat adalah kronologi peristiwa lanjutan pada kasus yang sama, sehingga perkembangan kasus tetap tercatat dalam satu laporan.

### Isian formulir

| Field                 | Wajib | Keterangan                       |
| --------------------- | ----- | -------------------------------- |
| Instansi              | Ya    | Instansi tempat siswa terdaftar. |
| Korban                | Ya    | Siswa yang terdaftar.            |
| Tanggal Kejadian      | Ya    |                                  |
| Lokasi                | Ya    | -                                |
| Pelapor               | Ya    | -                                |
| Nama Pelaku           | Ya    | Siswa yang terdaftar.            |
| Jenis Perundungan     | Ya    |                                  |
| Tindakan yang diambil | Ya    | -                                |
| Saksi                 | Tidak | -                                |
| Deskripsi Kejadian    | Ya    | -                                |
| Bukti Laporan         | Tidak | -                                |

## Hapus Perundungan

Link video:

Hapus data perundungan yang tidak valid. Data perundungan adalah laporan kasus beserta riwayat peristiwanya. Data yang dihapus tidak lagi tampil di daftar laporan perundungan. Aksi ini biasanya dipakai saat laporan tidak valid atau salah input.

<Callout kind="alert" collapsed="false">
  Penghapusan data perundungan bersifat permanen dan tidak dapat dikembalikan.
</Callout>
