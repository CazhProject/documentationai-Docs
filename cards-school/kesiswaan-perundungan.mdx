Menu Perundungan pada modul Kesiswaan digunakan admin sekolah untuk memantau dan mengelola daftar kasus [perundungan](/referensi/glosarium#perundungan) yang terjadi di sekolah.

<Callout kind="info" collapsed="false">
  Pelaporan juga dapat dilakukan melalui aplikasi. Lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).
</Callout>

### Informasi yang ditampilkan

Filter yang tersedia: **Pilih Instansi, Jenis Perundungan, Status Laporan&#x20;**(Sedang Pengajuan / Diproses / Selesai / Semua status)**,** dan **Tanggal Kejadian**. Jumlah baris per halaman dapat dipilih : 10, 50, 100, 500, atau 1000.

**Tabel Perundungan** menampilkan kolom: Kolom centang, Tanggal Kejadian, Korban, Pelapor, Pelaku, Lokasi, Jenis Perundungan, Status, Detail data perundungan (ikon mata pada kolom sebelum ikon tempat sampah (kolom paling kanan)), dan Hapus data perundungan (ikon tempat sampah pada kolom paling kanan).&#x20;

## Tambah Riwayat Perundungan

Link video:&#x20;

Fitur Tambah Riwayat Perundungan digunakan untuk admin menambahkan peristiwa perundungan terbaru.

### Isian formulir Riwayat Perundungan

| **Field**                 | **Wajib** | **Keterangan**                            |
| ------------------------- | --------- | ----------------------------------------- |
| **Instansi**              | Ya        | Instansi tempat siswa terdaftar.          |
| **Korban**                | Ya        | Siswa terdaftar.                          |
| **Tanggal Kejadian**      | Ya        | -                                         |
| **Lokasi**                | Ya        | -                                         |
| **Pelapor**               | Ya        | Siswa terdaftar.                          |
| **Nama Pelaku**           | Ya        | Siswa terdaftar.                          |
| **Jenis Perundungan**     | Ya        | -                                         |
| **Tindakan yang diambil** | Ya        | -                                         |
| **Saksi&#x20;**           | Tidak     | Nama saksi perundungan.                   |
| **Deskripsi Kejadian**    | Ya        | -                                         |
| **Bukti Laporan**         | Tidak     | File format JPEG, JPG, PNG maksimal 1 MB. |
