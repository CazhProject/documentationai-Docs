Menu Perundungan pada modul Kesiswaan memungkinkan admin memantau dan mengelola daftar kasus [perundungan](/referensi/glosarium#perundungan) yang terjadi di sekolah. Pelaporan juga dapat dilakukan melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Perundungan

Fitur List Perundungan memungkinkan admin melihat daftar data kasus perundungan yang terjadi pada suatu sekolah, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perundungan**.
3. Sistem akan menampilkan halaman List Perundungan.
4. Pengguna dapat menemukan data perundungan dengan menggunakan fitur **Pencarian** dan **Filter**.
5. Data list perundungan dapat disimpan pada perangkat pengguna dengan klik **Excel**.

## Detail Perundungan

Fitur Detail Perundungan memungkinkan admin melihat detail informasi perundungan, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perundungan**.
3. Kemudian klik ikon **Detail**.
4. Sistem akan menampilkan halaman Detail Perundungan.

## Tambah Riwayat Perundungan

Fitur Tambah Riwayat Perundungan digunakan admin untuk menambahkan peristiwa perundungan terbaru, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perundungan**.
3. Sistem akan menampilkan halaman Laporan Perundungan.
4. Klik **Tambah Riwayat**.
5. Isi form dengan sesuai dan benar, lalu klik **Simpan**.

## Hapus Perundungan

Fitur Hapus Perundungan digunakan admin untuk menghapus data perundungan yang tidak valid, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perundungan**.
3. Sistem akan menampilkan halaman Laporan Perundungan.
4. Geser laporan ke arah kanan dan klik ikon **Hapus**.
5. Untuk menghapus, klik **Hapus**.
