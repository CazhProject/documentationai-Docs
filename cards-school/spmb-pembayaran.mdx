# SPMB — Pembayaran

Menu Pembayaran SPMB digunakan Admin untuk memantau seluruh transaksi pembayaran [SPMB](/referensi/glosarium#spmb), baik per calon peserta didik maupun rekap agregat.

## Laporan Pembayaran

Admin memiliki kemampuan untuk mengelola laporan pembayaran:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Pembayaran SPMB**.
4. Klik tab **Laporan Pembayaran**.
5. Sistem akan menampilkan seluruh daftar laporan pembayaran calon peserta didik.
6. Pada daftar laporan pembayaran, Admin dapat:
   - Menemukan data pembayaran dengan cepat melalui fitur **Filter** dan **Pencarian**.
   - **Cetak** bukti pembayaran setiap calon peserta didik.
   - **Export** laporan pembayaran menjadi file Excel.

Untuk konfirmasi pembayaran tunai atau mengubah metode pembayaran per pendaftar, lihat [SPMB — Data Pendaftar](/cards-school/spmb-data-pendaftar#konfirmasi-pembayaran).

## Rekap Pembayaran

Admin memiliki kemampuan untuk mengelola data rekap pembayaran SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Pilih menu **SPMB**.
3. Pilih menu **Pembayaran SPMB**.
4. Klik tab **Rekapitulasi**.
5. Pada halaman Rekapitulasi, Admin dapat:
   - Melihat daftar rekapitulasi pembayaran setiap jurusan pada instansi.
   - **Export** data rekapitulasi menjadi file Excel.
