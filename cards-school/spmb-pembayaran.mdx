Menu Pembayaran SPMB digunakan admin sekolah untuk memantau seluruh transaksi pembayaran [SPMB](/referensi/glosarium#spmb), baik per calon peserta didik maupun rekap agregat per jurusan.

## Laporan Pembayaran

Memantau dan mengelola laporan pembayaran per calon peserta didik.

<Steps>
  <Step title="Buka Pembayaran SPMB">Login ke website, klik menu **Anggota**, pilih menu **SPMB**, lalu pilih menu **Pembayaran SPMB**.</Step>
  <Step title="Buka tab Laporan Pembayaran">Klik tab **Laporan Pembayaran**. Sistem akan menampilkan seluruh daftar laporan pembayaran calon peserta didik.</Step>
  <Step title="Kelola laporan">Temukan data pembayaran dengan cepat melalui fitur **Filter** dan **Pencarian**, **Cetak** bukti pembayaran setiap calon peserta didik, atau **Export** laporan pembayaran menjadi file Excel.</Step>
</Steps>

<Callout kind="tip">Untuk konfirmasi pembayaran tunai atau mengubah metode pembayaran per pendaftar, lihat [SPMB — Data Pendaftar](/cards-school/spmb-data-pendaftar#konfirmasi-pembayaran).</Callout>

## Rekap Pembayaran

Melihat rekapitulasi pembayaran SPMB per jurusan.

<Steps>
  <Step title="Buka Pembayaran SPMB">Login ke website, klik menu **Anggota**, pilih menu **SPMB**, lalu pilih menu **Pembayaran SPMB**.</Step>
  <Step title="Buka tab Rekapitulasi">Klik tab **Rekapitulasi**.</Step>
  <Step title="Kelola rekap">Lihat daftar rekapitulasi pembayaran setiap jurusan pada instansi, atau **Export** data rekapitulasi menjadi file Excel.</Step>
</Steps>
