[Cazhbox](/referensi/glosarium#cazhbox) adalah dompet saldo instansi pada ekosistem Cazh. Melalui menu ini Admin dapat memantau riwayat transaksi, memindahkan dana antara Cazhbox instansi dan Cazhbox pusat, mengisi saldo melalui Virtual Account, serta menarik saldo ke rekening instansi. Saldo Cazhbox juga menjadi salah satu sumber dana untuk [isi Kartu](/cards-school/kartu#isi-kartu-satuan) siswa; orang tua dapat melakukan top-up saldo anak melalui aplikasi [CARDS Parent](/cards-parent/top-up-saldo).

## Saldo Cazhbox Instansi

Admin dapat mengelola Cazhbox dengan melakukan penambahan atau penarikan dana dari saldo Cazhbox setiap instansi, serta melakukan penarikan ke rekening instansi melalui langkah-langkah berikut:

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Cazhbox**.
3. Sistem akan menampilkan halaman Cazhbox.
4. Di halaman ini, Anda dapat melihat riwayat transaksi.
5. Halaman ini juga memungkinkan Anda untuk melakukan penarikan dan penambahan saldo Cazhbox ke instansi pusat.
6. Selain itu, Anda dapat melakukan penarikan saldo ke rekening instansi.

## Topup Saldo Cazhbox dari Cazhbox Pusat

Admin dapat mengisi Cazhbox instansi dengan sumber dana dari Cazhbox pusat:

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Cazhbox**.
3. Sistem akan menampilkan halaman Cazhbox.
4. Pilih Cazhbox instansi yang akan diisi.
5. Klik tombol **Topup dari Pusat**.
6. Input nominal, klik tombol **Topup**.

## Tarik Saldo Cazhbox ke Cazhbox Pusat

Admin dapat menarik saldo dari Cazhbox instansi ke akun Cazhbox pusat:

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Cazhbox**.
3. Sistem akan menampilkan halaman Cazhbox.
4. Pilih Cazhbox instansi yang saldonya akan ditarik.
5. Klik tombol **Tarik ke Pusat**.
6. Input nominal, klik tombol **Tarik ke Pusat**.

## Topup Saldo Cazhbox Instansi

Admin dapat mengisi Cazhbox instansi dengan menggunakan nomor VA dari berbagai bank:

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Cazhbox**.
3. Sistem akan menampilkan halaman Cazhbox.
4. Pilih Cazhbox instansi yang akan diisi.
5. Klik tombol **Isi Saldo**.
6. Pilih nomor VA yang akan digunakan.
7. Isi saldo pada nomor VA tersebut menggunakan ATM atau mobile banking.

## Tarik Saldo Cazhbox Instansi

Admin dapat menarik saldo Cazhbox instansi ke nomor rekening instansi dengan biaya tertentu:

1. Login ke website, lalu klik menu **Keuangan**.
2. Klik submenu **Cazhbox**.
3. Sistem akan menampilkan halaman Cazhbox.
4. Pilih Cazhbox akun pusat.
5. Klik tombol **Tarik Saldo**.
6. Pilih **Rekening Penerima**, input **Nominal Penarikan**.
7. Klik tombol **Tarik Saldo**.
