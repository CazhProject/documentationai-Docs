Menu Perizinan pada modul Kesiswaan digunakan admin untuk mengelola dan memantau data [perizinan](/referensi/glosarium#perizinan) siswa yang tersimpan dalam sistem. Batas waktu pengajuan izin diatur melalui [Pengaturan Kesiswaan — Waktu Perizinan](/cards-school/pengaturan-lainnya#waktu-perizinan). Pengajuan izin oleh siswa/orang tua dilakukan melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Perizinan

Fitur Perizinan List digunakan untuk mengelola dan memantau data perizinan yang tersimpan dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perizinan**.
3. Sistem menampilkan data Perizinan.
4. Pada halaman Perizinan List, admin dapat:
   - Melihat daftar data perizinan yang tercatat dalam sistem.
   - Melakukan pencarian data perizinan menggunakan fitur **Search**.
   - Mengekspor data perizinan menggunakan fitur **Export**.
   - Memantau dan menelusuri informasi perizinan yang tersedia dalam sistem.

## Edit Perizinan

Fitur Edit Perizinan digunakan untuk memperbarui atau mengubah informasi perizinan yang telah tercatat dalam sistem, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perizinan**.
3. Sistem akan menampilkan halaman Perizinan.
4. Geser tabel ke sebelah kanan tabel.
5. Kemudian pilih **Edit**.
6. Edit data perizinan dengan sesuai, lalu klik **Simpan Perubahan**.

## Hapus Perizinan

Fitur Hapus Perizinan digunakan admin untuk menghapus data perizinan yang tidak valid, dengan langkah-langkah berikut:

1. Login ke website, kemudian klik menu **Kesiswaan**.
2. Kemudian klik submenu **Perizinan**.
3. Sistem akan menampilkan halaman Ringkasan Data Perizinan.
4. Geser tabel ke arah kanan dan klik ikon **Hapus**.
5. Untuk menghapus, klik **Hapus**.
