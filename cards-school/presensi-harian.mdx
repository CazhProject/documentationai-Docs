Menu Presensi Harian pada modul Akademik digunakan admin untuk mengelola kehadiran harian siswa, guru, dan staff — mulai dari halaman presensi via link, log presensi, data keterlambatan, rekap, hingga pengaturannya. Kehadiran siswa dapat dipantau orang tua melalui aplikasi CARDS Parent — lihat [Akademik di CARDS Parent](/cards-parent/akademik).

## Link Presensi Harian

Sistem menampilkan halaman presensi harian yang dapat diakses melalui link. Pada halaman ini akan ditampilkan daftar anggota yang telah melakukan presensi serta QR presensi.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Presensi Harian**.
3. Klik submenu **Pengaturan**.
4. Sistem akan menampilkan halaman Presensi Harian dari masing-masing [instansi](/referensi/glosarium#instansi).
5. Klik **Link Presensi**.
6. Sistem akan menampilkan halaman **Presensi Harian**.
7. Lakukan presensi anggota dengan salah satu cara berikut:
   - Masukkan nomor kartu atau [RFID](/referensi/glosarium#rfid) anggota pada menu tersebut.
   - Tampilkan QR code untuk discan anggota yang akan presensi (lihat [QR Presensi di CARDS Edu](/cards-edu/qr)).
   - Gunakan kamera perangkat untuk scan kartu anggota.
8. Pilih data anggota yang muncul; anggota otomatis akan terdaftar telah melakukan presensi.

## Log Presensi Harian

Admin dapat mengakses data log presensi harian untuk setiap siswa, guru, dan staff melalui langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Log**.
4. Sistem akan menampilkan **Data Log Presensi Harian**.

### Edit Log Presensi Harian

Admin dapat mengubah data log presensi harian untuk setiap siswa, guru, dan staff melalui langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Log**.
4. Sistem akan menampilkan **Data Log Presensi Harian**.
5. Klik tombol **Edit**.
6. Sistem menampilkan **Form Edit**.
7. Isi form edit log presensi harian.
8. Klik **Simpan**.
9. Sistem akan menyimpan data.

## Terlambat

### Melihat Data Presensi Terlambat

Admin memiliki kemampuan untuk melihat presensi siswa yang terlambat dan dapat menambahkan data untuk siswa tersebut:

1. Masuk ke website, lalu klik menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Terlambat**.
4. Sistem akan menampilkan **Data Presensi Terlambat**.

### Tambah Data Terlambat

Admin memiliki kemampuan untuk memantau kehadiran siswa yang terlambat serta menambahkan informasi terkait siswa tersebut:

1. Masuk ke situs web, lalu pilih menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Terlambat**.
4. Sistem akan menampilkan **Data Presensi Terlambat**.
5. Klik tombol **Tambah Terlambat**.
6. Lengkapi formulir **Tambah**.
7. Klik tombol **Simpan**.
8. Data terlambat akan berhasil dibuat.

## Rekap Siswa

Admin dapat melihat data rekap presensi harian setiap siswa.

1. Masuk ke situs web, lalu pilih menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Rekap**.
4. Klik submenu **Siswa**.
5. Sistem akan menampilkan **Data Rekap Presensi Harian Siswa**.
6. Admin dapat mengelola data rekap presensi harian siswa seperti:
   - Export data rekap.
   - Detail data rekap → menambah manual data rekap.

### Detail Rekap Siswa

Admin dapat melihat detail rekap presensi harian masing-masing siswa secara lengkap. Halaman ini akan menampilkan tanggal presensi, status presensi, waktu presensi, serta keterangan apabila izin/sakit.

1. Masuk ke situs web, lalu pilih menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Rekap**.
4. Klik submenu **Siswa**.
5. Sistem akan menampilkan **Data Rekap Presensi Harian Siswa**.
6. Klik aksi **Detail**.

### Edit Presensi Manual Siswa

Admin dapat mengedit data presensi setiap siswa jika terjadi penyesuaian presensi harian.

1. Masuk ke situs web, lalu pilih menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Rekap**.
4. Klik submenu **Siswa**.
5. Sistem akan menampilkan **Data Rekap Presensi Harian Siswa**.
6. Klik aksi **Detail**.
7. Pilih tanggal presensi yang akan diubah.
8. Klik aksi **Edit**, isi formulir.
9. Klik **Simpan Perubahan**.

### Tambah Presensi Manual Siswa

Admin dapat menambah data presensi secara manual pada setiap siswa jika terjadi penyesuaian presensi harian.

1. Masuk ke situs web, lalu pilih menu **Akademik**.
2. Pilih submenu **Presensi Harian**.
3. Klik submenu **Rekap**.
4. Klik submenu **Siswa**.
5. Sistem akan menampilkan **Data Rekap Presensi Harian Siswa**.
6. Klik aksi **Detail**.
7. Klik tombol **Tambah Presensi Manual**, isi formulir.
8. Klik **Simpan Perubahan**.

## Pengaturan Presensi Harian

Pengaturan Presensi Harian memungkinkan admin sekolah mengelola pengaturan jadwal presensi harian:

1. Klik menu **Akademik**.
2. Klik submenu **Presensi Harian**.
3. Klik submenu **Pengaturan**.
4. Klik **Detail** salah satu instansi.
5. Sistem akan menampilkan list pengaturan presensi harian.

### Detail Pengaturan

Detail Pengaturan Presensi Harian memungkinkan admin sekolah mengelola pengaturan jadwal presensi harian:

1. Klik menu **Akademik**.
2. Klik submenu **Presensi Harian**.
3. Klik submenu **Pengaturan**.
4. Sistem akan menampilkan list pengaturan presensi harian.
5. Klik salah satu detail instansi dengan klik tombol **>**.
6. Sistem akan menampilkan detail pengaturan presensi harian instansi yang dipilih.

### Sistem Shift Staff

Presensi Harian Sistem Shift Staff memungkinkan admin sekolah mengelola pengaturan jadwal presensi harian staff berdasarkan shift yang berlaku. Langkah-langkahnya sebagai berikut:

1. Klik menu **Akademik**.
2. Klik submenu **Presensi Harian**.
3. Klik submenu **Pengaturan**.
4. Sistem akan menampilkan list pengaturan presensi harian.
5. Klik salah satu detail instansi dengan klik tombol **>**.
6. Sistem akan menampilkan detail pengaturan presensi harian instansi yang dipilih.
7. Aktifkan toggle **Waktu presensi siswa, guru & staff berbeda**.
8. Aktifkan toggle **Sistem Shift** pada presensi Staff.
9. Input data, lalu klik tombol **Simpan**.

### Max Keterlambatan

Pengaturan Max Keterlambatan memungkinkan admin menentukan batas maksimal keterlambatan yang diperbolehkan dalam presensi harian untuk seluruh pengguna yang menggunakan sistem presensi. Langkah-langkahnya sebagai berikut:

1. Klik menu **Akademik**.
2. Klik submenu **Presensi Harian**.
3. Klik submenu **Pengaturan**.
4. Sistem akan menampilkan list pengaturan presensi harian.
5. Klik salah satu detail instansi dengan klik tombol **>**.
6. Sistem akan menampilkan detail pengaturan presensi harian instansi yang dipilih.
7. Aktifkan toggle **Waktu presensi siswa, guru & staff berbeda**.
8. Aktifkan toggle **Max Keterlambatan**.
9. Input data, lalu klik tombol **Simpan**.
