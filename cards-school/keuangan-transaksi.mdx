Menu Transaksi digunakan Admin untuk memonitor transaksi tagihan instansi dan melakukan pembayaran tagihan anggota, baik satuan maupun massal. Pembayaran menggunakan [Kartu](/referensi/glosarium#kartu) juga dapat dilakukan di kasir melalui [CazhPOS](/cazhpos/saldo-dan-kartu).

## Daftar Transaksi Terakhir

Admin memiliki kemampuan untuk mengelola transaksi tagihan. Langkah untuk memonitor transaksi tagihan anggota:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Transaksi**.
3. Sistem akan menampilkan halaman Daftar Transaksi Tagihan Terakhir yang terjadi pada instansi.
4. Admin dapat mengakses detail transaksi terakhir secara langsung dengan klik ikon **Detail**.
5. Untuk menampilkan daftar transaksi tagihan milik anggota tertentu, Admin dapat menggunakan fitur pencarian berdasarkan **Nomor Kartu** atau **Nama**; kemudian sistem akan menampilkan data tagihan anggota tersebut.

## Bayar Tagihan Satuan

Admin memiliki akses untuk melakukan pembayaran tagihan anggota. Langkah melakukan pembayaran tagihan secara satuan:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Transaksi**.
3. Gunakan fitur pencarian dengan memasukkan **Nomor Kartu** atau **Nama** untuk menemukan anggota secara spesifik.
4. Sistem akan menampilkan data tagihan milik anggota tersebut.
5. Klik **Aksi** pada tagihan yang akan dibayarkan.
6. Pilih metode pembayaran yang diinginkan:
   - **Buat Invoice Tagihan** → digunakan untuk pembayaran melalui Virtual Account Bank. Invoice dapat dikirimkan kepada anggota melalui WhatsApp.
   - **Tunai**.
   - **Kartu** → memerlukan verifikasi PIN [Kartu](/referensi/glosarium#kartu).
7. Masukkan nominal pembayaran:
   - Nominal penuh hingga tagihan berstatus **Lunas**.
   - Nominal sebagian sebagai pembayaran cicilan.
8. Klik **Simpan**, sistem akan memproses pembayaran tagihan.

## Bayar Tagihan Massal

Admin memiliki akses untuk melakukan pembayaran tagihan anggota. Langkah melakukan pembayaran tagihan secara massal:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Transaksi**.
3. Gunakan fitur pencarian dengan memasukkan **Nomor Kartu** atau **Nama** untuk menemukan anggota secara spesifik.
4. Sistem akan menampilkan data tagihan milik anggota tersebut.
5. Pilih beberapa tagihan yang akan dibayarkan menggunakan checkbox.
6. Pilih metode pembayaran yang diinginkan:
   - **Bayar Sekaligus** → digunakan untuk pembayaran melalui Virtual Account Bank atau Tunai. Invoice dapat dikirimkan kepada anggota melalui WhatsApp.
   - **Bayar Kartu** → memerlukan verifikasi PIN Kartu.
7. Masukkan nominal pembayaran. Pembayaran massal hanya dapat dilakukan dengan nominal penuh (lunas), tidak dapat dicicil.
8. Klik **Simpan**, sistem akan memproses pembayaran tagihan.

## Detail Pembayaran Tagihan

Admin memiliki akses untuk memonitor detail pembayaran tagihan. Langkah akses detail pembayaran tagihan:

1. Login ke website, lalu klik menu **Keuangan**.
2. Kemudian, klik submenu **Transaksi**.
3. Gunakan fitur pencarian dengan memasukkan **Nomor Kartu** atau **Nama** untuk menemukan anggota secara spesifik.
4. Sistem akan menampilkan data tagihan milik anggota tersebut.
5. Klik **Aksi** pada tagihan yang akan dilihat detail pembayarannya.
6. Pilih **Detail Pembayaran**.
7. Sistem akan menampilkan detail pembayaran tagihan yang dipilih:
   - Jika status tagihan **Terbayar Sebagian** atau **Lunas**, sistem menampilkan daftar pembayaran yang telah dilakukan. Admin dapat melihat bukti pembayaran dengan klik pada nomor **Ref ID** pada transaksi pembayaran yang diinginkan.
   - Jika status tagihan **Belum Terbayar**, sistem hanya menampilkan informasi tagihan tanpa daftar pembayaran karena belum terdapat transaksi pembayaran yang tercatat.
