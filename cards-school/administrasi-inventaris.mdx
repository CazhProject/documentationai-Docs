---
title: "Administrasi: Inventaris"
description: "Panduan mengelola inventaris barang di CARDS School — melihat, menambah, mengubah, dan menghapus data barang serta riwayat inventaris."
---

Menu Inventaris digunakan Admin untuk mencatat dan memantau seluruh barang milik instansi beserta riwayat pergerakannya. Terdapat dua submenu utama: **Barang** dan **Riwayat**.

## List Barang

Admin memiliki kemampuan untuk mengelola inventaris barang dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Inventaris**.
4. Pilih menu **Barang**.
5. Sistem akan menampilkan halaman Barang.
6. Pada halaman tersebut, Admin dapat:
   - Melakukan filter data barang.
   - Mencari data barang.
   - Mengekspor data barang.

## Tambah Barang

Admin memiliki kemampuan untuk menambah inventaris barang dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Inventaris**.
4. Pilih menu **Barang**.
5. Sistem akan menampilkan halaman Barang Inventaris.
6. Klik opsi **Tambah Barang**.
7. Sistem akan menampilkan form tambah barang.
8. Isi form tambah barang sesuai dengan kebutuhan.
9. Klik **Simpan** untuk melanjutkan proses tambah barang.
10. Sistem otomatis akan menampilkan data barang yang ditambahkan.

## Edit Barang

Admin memiliki kemampuan untuk mengedit inventaris barang dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Inventaris**.
4. Pilih menu **Barang**.
5. Sistem akan menampilkan halaman Barang Inventaris.
6. Klik opsi **Edit Barang** pada data barang yang ingin diedit.
7. Sistem akan menampilkan form edit barang.
8. Isi form edit barang sesuai dengan kebutuhan.
9. Klik **Simpan** untuk melanjutkan proses edit barang.
10. Sistem otomatis akan memperbarui data barang yang diedit.

## Hapus Barang

Admin memiliki kemampuan untuk menghapus data barang inventaris dengan langkah berikut:

1. Login ke website.
2. Klik menu **Administrasi**.
3. Pilih submenu **Inventaris**.
4. Pilih menu **Barang**.
5. Sistem akan menampilkan halaman Barang Inventaris.
6. Klik tombol **Hapus** pada data barang yang ingin dihapus.
7. Klik tombol **Hapus** pada konfirmasi untuk menghapus data barang.
8. Sistem otomatis akan menghapus barang.

## List Riwayat Inventaris

Fitur list riwayat inventaris memungkinkan Admin untuk memonitor seluruh riwayat inventaris, dengan langkah-langkah sebagai berikut:

1. Login ke website, kemudian klik menu **Administrasi**.
2. Kemudian klik submenu **Inventaris**.
3. Kemudian klik submenu **Riwayat**.
4. Sistem akan menampilkan halaman Riwayat Inventaris.
5. Pada halaman Riwayat Inventaris, Admin dapat:
   - Melihat daftar riwayat inventaris yang telah tercatat.
   - Melakukan pencarian data riwayat inventaris menggunakan fitur Search.
   - Melakukan penyaringan data riwayat inventaris menggunakan fitur Filter.
   - Mengekspor data riwayat inventaris menggunakan fitur Export.

## Tambah Riwayat Inventaris

Fitur tambah riwayat inventaris memungkinkan Admin untuk menambah data, dengan langkah-langkah sebagai berikut:

1. Login ke website, kemudian klik menu **Administrasi**.
2. Kemudian klik submenu **Inventaris**.
3. Kemudian klik submenu **Riwayat**.
4. Sistem akan menampilkan halaman Riwayat Inventaris.
5. Klik **Tambah Riwayat Inventaris**.
6. Isi form dengan data yang sesuai dan benar, lalu klik **Simpan**.

## Edit Riwayat Inventaris

Fitur Edit Riwayat Inventaris digunakan untuk menyunting data yang sudah ada, dengan langkah-langkah sebagai berikut:

1. Login ke website, kemudian klik menu **Administrasi**.
2. Kemudian klik submenu **Inventaris**, lalu submenu **Riwayat**.
3. Sistem akan menampilkan halaman Riwayat Inventaris.
4. Klik ikon **Titik Tiga** yang berada pada sebelah kanan tabel.
5. Kemudian pilih **Edit**.
6. Edit data riwayat dengan sesuai dan klik **Simpan Perubahan**.

## Hapus Riwayat Inventaris

Fitur Hapus Riwayat Inventaris memungkinkan Admin untuk menghapus baris data yang tidak valid, dengan langkah-langkah sebagai berikut:

1. Login ke website, kemudian klik menu **Administrasi**.
2. Kemudian klik submenu **Inventaris**, lalu submenu **Riwayat**.
3. Sistem akan menampilkan halaman Riwayat Inventaris.
4. Pilih baris yang ingin dihapus dengan cara mencentang kolom paling kiri.
5. Kemudian pilih **Hapus yang Dipilih** pada bagian atas tabel.
6. Klik konfirmasi dan baris data akan terhapus.
