Menu Inventaris digunakan admin sekolah untuk mencatat dan memantau seluruh barang milik instansi beserta riwayat pergerakannya. Terdapat dua submenu utama: **Barang** dan **Riwayat**, diakses melalui menu **Administrasi** → submenu **Inventaris**.

## List Barang

Melihat dan mengelola daftar barang inventaris.

### Informasi yang ditampilkan

Tabel Barang menampilkan kolom: **Instansi**, **Dibuat oleh**, **Kategori**, **Nama**, **Foto**, **Barcode**, **Masuk**, **Keluar**, **Harga**, dan **Total Harga**. Filter tersedia untuk **Instansi**, **Operator**, dan **Kategori**. Export Excel menghasilkan file **Daftar Barang Inventaris**. Kategori barang dikelola melalui menu **Pengaturan** → **Inventaris** (isian: Instansi dan Nama Kategori).

## Tambah Barang

Mendaftarkan barang milik instansi ke dalam catatan inventaris agar keberadaan, jumlah, dan nilainya dapat dipantau. Barang yang ditambahkan langsung tampil di daftar Barang.

### Isian formulir

| Isian                                                 | Ketentuan                                                |
| ----------------------------------------------------- | -------------------------------------------------------- |
| Instansi                                              | Wajib.                                                   |
| Foto                                                  | Opsional, gambar jpeg/jpg/png/heic/heif, maksimal 10 MB. |
| Kategori                                              | Wajib.                                                   |
| Nama Barang                                           | Wajib, maksimal 255 karakter.                            |
| Harga                                                 | Wajib, angka, minimal 0.                                 |
| Lokasi                                                | Opsional, maksimal 255 karakter.                         |
| Kode Barang                                           | Opsional.                                                |
| No. Registrasi                                        | Opsional.                                                |
| PIC                                                   | Opsional (dipilih dari anggota).                         |
| Tipe Mesin, Nomor Mesin/Rangka, Tahun Pembuatan, Merk | Opsional (untuk barang bermesin).                        |

## Edit Barang

Memperbarui data barang inventaris, misalnya memperbaiki nama, harga, lokasi, atau mencatat kondisi barang (digunakan, tidak digunakan, rusak, hilang). Perubahan menggantikan data lama pada barang yang dipilih.

<Callout kind="info" collapsed="false">
  Pada form edit, kondisi barang dicatat dalam empat kategori jumlah: **Digunakan**, **Tidak Digunakan**, **Rusak**, dan **Hilang** (masing-masing wajib, angka, minimal 0). Total keempat kondisi tidak boleh melebihi stok tersedia; jika melebihi, sistem menolak dengan pesan "Kondisi barang melebihi stock tersedia!".
</Callout>

## Hapus Barang

Mengeluarkan barang dari catatan inventaris, misalnya barang yang salah input atau sudah tidak dimiliki instansi. Barang yang dihapus tidak lagi tampil di daftar Barang.

Data barang yang dihapus tidak dapat dikembalikan. Pastikan barang yang dipilih sudah benar sebelum mengonfirmasi.

## List Riwayat Inventaris

Memonitor seluruh riwayat inventaris.

### Informasi yang ditampilkan

Tabel Riwayat Inventaris menampilkan kolom: **Instansi**, **Tanggal**, **Barang**, **Stok**, **Keterangan** (jenis riwayat), dan **Status**, dengan detail tambahan Catatan dan Info Lain. Filter tersedia untuk **Instansi**, **Status**, **Keterangan**, dan **Barang**. Export menghasilkan file **Riwayat Inventaris**.

<Callout kind="info" collapsed="false">
  Riwayat inventaris berbasis pergerakan stok dengan dua status: **Masuk** (stok bertambah) dan **Keluar** (stok berkurang).
</Callout>

## Tambah Riwayat Inventaris

Mencatat pergerakan stok suatu barang, yaitu **Masuk** (stok bertambah, misalnya pembelian) atau **Keluar** (stok berkurang, misalnya dipinjam atau dihapuskan), sehingga jumlah stok barang pada inventaris tetap akurat.

## Edit Riwayat Inventaris

Memperbaiki catatan pergerakan stok yang keliru, misalnya salah jumlah atau salah keterangan, tanpa perlu menghapus dan membuat ulang catatannya.

## Hapus Riwayat Inventaris

Menghapus baris catatan pergerakan stok yang tidak valid, misalnya salah input yang tidak bisa diperbaiki lewat edit. Baris yang dihapus tidak lagi tampil di daftar riwayat.

Baris riwayat yang dihapus tidak dapat dikembalikan.
