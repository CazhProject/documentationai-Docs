Menu Presensi Mapel pada modul Akademik digunakan admin untuk memantau kehadiran siswa per [mata pelajaran](/referensi/glosarium#mata-pelajaran). Data presensi ini juga dapat dipantau orang tua melalui aplikasi CARDS Parent — lihat [Akademik di CARDS Parent](/cards-parent/akademik).

## Rekap Siswa

Admin dapat melihat list nama siswa; sistem akan menampilkan pelajaran yang diikuti siswa tersebut pada detail masing-masing siswa.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Presensi Mapel**.
3. Klik submenu **Rekap Siswa**.
4. Sistem akan menampilkan **List Siswa**.

## Input Presensi Mata Pelajaran

Fitur ini digunakan untuk melakukan input presensi mata pelajaran.

1. Login ke website, lalu klik menu **Akademik**.
2. Klik submenu **Presensi Mata Pelajaran**.
3. Sistem akan menampilkan kolom untuk input nama atau nomor kartu guru.
4. Input nama atau nomor kartu guru.

Siswa juga dapat melakukan presensi menggunakan QR di aplikasi — lihat [QR Presensi di CARDS Edu](/cards-edu/qr).
