Menu Pelanggaran pada modul Kesiswaan digunakan admin untuk mencatat dan mengelola pelanggaran siswa. Tipe pelanggaran yang tersedia dikelola melalui [Pengaturan Kesiswaan — Tipe Pelanggaran](/cards-school/pengaturan-lainnya#tipe-pelanggaran). Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Pelanggaran

Admin memiliki kemampuan untuk mengelola data pelanggaran siswa. Langkah-langkahnya:

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Pelanggaran**.
4. Sistem akan menampilkan halaman pelanggaran siswa.
5. Pada halaman tersebut, admin dapat:
   - Melihat daftar data pelanggaran.
   - Mencari data pelanggaran.
   - Export data pelanggaran.

## Tambah Pelanggaran

Admin memiliki kemampuan untuk menambahkan data pelanggaran siswa ke dalam sistem.

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Pelanggaran**.
4. Sistem akan menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.
5. Klik tombol **Tambah Pelanggaran**.
6. Sistem akan menampilkan form tambah pelanggaran siswa.
7. Isi data pelanggaran sesuai kebutuhan.
8. Klik tombol **Simpan** untuk menyimpan data pelanggaran siswa.

## Edit Pelanggaran

Admin memiliki kemampuan untuk mengubah dan memperbarui data pelanggaran siswa yang telah tersimpan pada sistem.

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Pelanggaran**.
4. Sistem akan menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.
5. Cari dan pilih data pelanggaran yang ingin diubah.
6. Klik opsi **Edit** pada data pelanggaran yang dipilih.
7. Sistem akan menampilkan form edit data pelanggaran.
8. Ubah data pelanggaran sesuai kebutuhan.
9. Klik tombol **Simpan** untuk menyimpan perubahan data pelanggaran siswa.

## Hapus Pelanggaran

Admin memiliki kemampuan untuk menghapus data pelanggaran siswa.

1. Login menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih submenu **Pelanggaran**.
4. Sistem akan menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.
5. Pilih tombol **Hapus**.
6. Klik tombol **Hapus** untuk melanjutkan proses penghapusan data pelanggaran siswa.
