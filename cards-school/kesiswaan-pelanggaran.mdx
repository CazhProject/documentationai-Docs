Menu Pelanggaran pada modul Kesiswaan digunakan admin sekolah untuk mencatat dan mengelola pelanggaran siswa.

<Callout kind="info">
Tipe pelanggaran yang tersedia dikelola melalui [Pengaturan Kesiswaan — Tipe Pelanggaran](/cards-school/pengaturan-lainnya#tipe-pelanggaran). Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).
</Callout>

## List Pelanggaran

Lihat, cari, dan ekspor daftar data pelanggaran siswa.

<Steps>
  <Step title="Buka menu Kesiswaan">Login menggunakan akun Admin, lalu buka menu **Kesiswaan**.</Step>
  <Step title="Buka Pelanggaran">Pilih submenu **Pelanggaran**. Sistem menampilkan halaman pelanggaran siswa.</Step>
  <Step title="Kelola daftar">Pada halaman tersebut, admin dapat:\n    - Melihat daftar data pelanggaran.\n    - Mencari data pelanggaran.\n    - Export data pelanggaran.</Step>
</Steps>

## Tambah Pelanggaran

Catat data pelanggaran siswa baru ke dalam sistem.

<Steps>
  <Step title="Buka Pelanggaran">Login menggunakan akun Admin, buka menu **Kesiswaan**, lalu pilih submenu **Pelanggaran**. Sistem menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.</Step>
  <Step title="Buka form tambah">Klik tombol **Tambah Pelanggaran**. Sistem menampilkan form tambah pelanggaran siswa.</Step>
  <Step title="Isi data">Isi data pelanggaran sesuai kebutuhan.</Step>
  <Step title="Simpan">Klik tombol **Simpan** untuk menyimpan data pelanggaran siswa.</Step>
</Steps>

## Edit Pelanggaran

Ubah dan perbarui data pelanggaran siswa yang telah tersimpan.

<Steps>
  <Step title="Buka Pelanggaran">Login menggunakan akun Admin, buka menu **Kesiswaan**, lalu pilih submenu **Pelanggaran**. Sistem menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.</Step>
  <Step title="Pilih data">Cari dan pilih data pelanggaran yang ingin diubah, lalu klik opsi **Edit** pada data yang dipilih. Sistem menampilkan form edit data pelanggaran.</Step>
  <Step title="Ubah data">Ubah data pelanggaran sesuai kebutuhan.</Step>
  <Step title="Simpan">Klik tombol **Simpan** untuk menyimpan perubahan data pelanggaran siswa.</Step>
</Steps>

## Hapus Pelanggaran

Hapus data pelanggaran siswa dari sistem.

<Steps>
  <Step title="Buka Pelanggaran">Login menggunakan akun Admin, buka menu **Kesiswaan**, lalu pilih submenu **Pelanggaran**. Sistem menampilkan halaman pelanggaran siswa beserta daftar data pelanggaran yang tersedia.</Step>
  <Step title="Hapus data">Pilih tombol **Hapus**.</Step>
  <Step title="Konfirmasi">Klik tombol **Hapus** untuk melanjutkan proses penghapusan data pelanggaran siswa.</Step>
</Steps>

<Callout kind="alert">
Penghapusan data pelanggaran bersifat permanen dan tidak dapat dikembalikan.
</Callout>
