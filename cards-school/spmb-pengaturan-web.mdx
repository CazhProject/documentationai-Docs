Menu Pengaturan Web digunakan admin sekolah untuk mengelola tampilan dan konfigurasi website [SPMB](/referensi/glosarium#spmb) (Sistem Penerimaan Murid Baru) lembaga. Seluruh pengaturan diakses melalui menu **Anggota** → submenu **SPMB** → menu **Pengaturan Web**, lalu pilih tab yang sesuai.

## Profil

Mengedit profil yang tampil pada landing page SPMB.

<Steps>
  <Step title="Buka Pengaturan Web">Login ke website, klik menu **Anggota**, klik submenu **SPMB**, lalu pilih menu **Pengaturan Web**.</Step>
  <Step title="Buka tab Profil">Sistem akan menampilkan tab **Profil**.</Step>
  <Step title="Ubah profil">Pada halaman tersebut, Admin dapat mengubah informasi profil.</Step>
</Steps>

## Beranda

Mengedit tampilan alur pendaftaran pada landing page SPMB.

<Steps>
  <Step title="Buka Pengaturan Web">Login ke website, klik menu **Anggota**, klik submenu **SPMB**, lalu pilih menu **Pengaturan Web**.</Step>
  <Step title="Buka tab Beranda">Klik tab **Beranda**.</Step>
  <Step title="Atur beranda">Setting alur pendaftaran menggunakan template yang telah disediakan, serta setting deskripsi dan animasi alur pendaftaran sesuai kebutuhan.</Step>
</Steps>

## Jadwal

Mengelola jadwal SPMB beserta kuota dan biayanya.

<Steps>
  <Step title="Buka Pengaturan Web">Login ke website, klik menu **Anggota**, pilih submenu **SPMB**, lalu pilih menu **Pengaturan Web**.</Step>
  <Step title="Buka tab Jadwal">Pilih menu navbar **Jadwal**.</Step>
  <Step title="Kelola jadwal">Lihat daftar jadwal SPMB yang telah dibuat, buat jadwal SPMB baru, hapus jadwal, atau edit jadwal SPMB.</Step>
</Steps>

Pada edit jadwal SPMB, Admin dapat:

- Setting kuota penerimaan.
- Setting biaya SPMB.
- Setting harga coret.
- Setting minimum pembayaran.

## Program Khusus

Mengelola jalur penerimaan dan sistem pendidikan.

<Steps>
  <Step title="Buka Pengaturan Web">Login ke website, klik menu **Anggota**, pilih submenu **SPMB**, lalu pilih menu **Pengaturan Web**.</Step>
  <Step title="Buka tab Program Khusus">Pilih menu navbar **Program Khusus**.</Step>
  <Step title="Kelola program">Kelola dua opsi yang tersedia sesuai tabel di bawah.</Step>
</Steps>

| Opsi | Kemampuan |
| --- | --- |
| **Jalur Penerimaan** | Melihat list jalur penerimaan, edit, hapus, dan tambah jalur penerimaan. |
| **Sistem Pendidikan** | Melihat daftar sistem pendidikan, edit, hapus, dan tambah daftar sistem pendidikan. |
