# SPMB — Pengaturan Web

Menu Pengaturan Web digunakan Admin untuk mengelola tampilan dan konfigurasi website [SPMB](/referensi/glosarium#spmb) (Sistem Penerimaan Murid Baru) lembaga. Seluruh pengaturan diakses melalui menu **Anggota** → submenu **SPMB** → menu **Pengaturan Web**.

## Profil

Admin memiliki kemampuan untuk mengedit profil pada landing page SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Klik submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Sistem akan menampilkan tab **Profil**.
5. Pada halaman tersebut, Admin dapat mengubah informasi profil.

## Beranda

Admin memiliki kemampuan untuk mengedit tampilan alur pendaftaran pada landing page SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Klik submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Klik tab **Beranda**.
5. Pada halaman tersebut, Admin dapat:
   - Setting alur pendaftaran menggunakan template yang telah disediakan.
   - Setting deskripsi dan animasi alur pendaftaran sesuai kebutuhan.

## Jadwal

Admin memiliki kemampuan untuk mengelola jadwal SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih menu navbar **Jadwal**.
5. Pada halaman tersebut, Admin dapat:
   - Melihat daftar jadwal SPMB yang telah dibuat.
   - Membuat jadwal SPMB baru.
   - Menghapus jadwal SPMB.
   - Edit jadwal SPMB:
     1. Setting kuota penerimaan.
     2. Setting biaya SPMB.
     3. Setting harga coret.
     4. Setting minimum pembayaran.

## Status Kustom

Admin memiliki kemampuan untuk mengelola status kustom untuk proses SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Klik navbar **Status Kustom**.
5. Pada halaman Status Kustom, Admin dapat:
   - Melihat daftar status kustom.
   - Menambah status kustom.
   - Edit status kustom.
   - Menghapus status kustom.

## Formulir Standar

Admin memiliki kemampuan untuk mengelola formulir standar yang ditampilkan pada calon peserta didik:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih menu navbar **Formulir Standar**.
5. Pada halaman Formulir Standar, Admin dapat:
   - Edit data formulir standar untuk ditampilkan atau disembunyikan.
   - Setting data formulir untuk wajib diisi atau tidak.
   - Setting data formulir ditampilkan pada semua jalur atau jalur tertentu.

## Program Khusus

Admin memiliki kemampuan untuk mengelola program khusus:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih menu navbar **Program Khusus**.
5. Halaman Program Khusus memiliki dua opsi:
   - **Jalur Penerimaan**: melihat list jalur penerimaan, edit jalur penerimaan, hapus jalur penerimaan, dan tambah jalur penerimaan.
   - **Sistem Pendidikan**: melihat daftar sistem pendidikan, edit daftar sistem pendidikan, hapus daftar sistem pendidikan, dan tambah daftar sistem pendidikan.

## Daftar Ulang

Admin memiliki kemampuan untuk mengelola [tagihan](/referensi/glosarium#tagihan) daftar ulang:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih submenu **Lainnya**.
5. Pilih menu **Daftar Ulang**.
6. Pada halaman Daftar Ulang, Admin dapat:
   - Melihat daftar tagihan daftar ulang.
   - Menambah tagihan daftar ulang.
   - Edit tagihan daftar ulang.
   - Hapus tagihan daftar ulang.
7. Tagihan akan ditambahkan otomatis kepada siswa yang dinyatakan **Diterima**.

## Kunjungi Website

Admin memiliki kemampuan untuk mengakses website SPMB secara langsung:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih submenu **Lainnya**.
5. Pilih menu **Kunjungi Website**.
6. Sistem akan redirect ke landing page website SPMB.

## Analytic

Admin memiliki kemampuan untuk memonitor analytic yang terjadi pada halaman website SPMB:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **SPMB**.
3. Pilih menu **Pengaturan Web**.
4. Pilih submenu **Lainnya**.
5. Pilih menu **Analytic**.
6. Pada halaman Analytic, Admin dapat melihat jumlah pengunjung website.
