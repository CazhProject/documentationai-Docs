# Anggota — Alumni

Menu Alumni digunakan untuk mengelola anggota yang sudah tidak aktif — baik alumni siswa maupun alumni guru. Anggota aktif dipindahkan ke alumni melalui aksi **Pindah Alumni** di halaman [Siswa](/cards-school/anggota-siswa#pindah-alumni) atau [Guru](/cards-school/anggota-guru#pindah-guru-ke-alumni).

## Pengelolaan Data Alumni Siswa

Admin memiliki kemampuan untuk mengelola data alumni siswa, termasuk mengubah status mereka menjadi siswa aktif kembali.

1. Login ke website, lalu klik menu **Anggota**.
2. Selanjutnya, pilih submenu **Alumni**.
3. Kemudian, klik submenu **Siswa**.
4. Sistem akan menampilkan **Halaman Siswa Alumni**.

## Jadikan Siswa Aktif

Admin memiliki kemampuan untuk menjadikan alumni menjadi siswa aktif kembali:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **Alumni**.
3. Pilih menu **Siswa**.
4. Pilih siswa yang akan diaktifkan.
5. Pilih **Ubah ke Siswa Aktif** → **Simpan**.
6. Pada halaman jadikan siswa aktif, Admin dapat mengubah status siswa alumni menjadi siswa aktif.

## Pengelolaan Data Alumni Guru

Admin memiliki kemampuan untuk mengelola guru alumni:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **Alumni**.
3. Pilih menu **Guru**.
4. Pada halaman Guru, Admin dapat:
   - Melihat list data alumni guru.
   - Export data.
   - Mencari data alumni guru.
   - Filter data alumni guru.

## Jadikan Guru Aktif

Admin memiliki kemampuan untuk menjadikan alumni menjadi guru aktif kembali:

1. Login ke website, kemudian klik menu **Anggota**.
2. Sistem akan menampilkan submenu **Alumni**.
3. Pilih menu **Guru**.
4. Pilih guru yang akan diaktifkan.
5. Pilih **Ubah ke Guru Aktif** → **Simpan**.
6. Pada halaman jadikan guru aktif, Admin dapat mengubah status guru alumni menjadi guru aktif.
