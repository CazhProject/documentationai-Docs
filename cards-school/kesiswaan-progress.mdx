Menu Progress Kegiatan pada modul Kesiswaan digunakan untuk memantau dan mengelola perkembangan kegiatan siswa. Data kesiswaan juga dapat diakses melalui aplikasi — lihat [Kesiswaan di CARDS Edu](/cards-edu/kesiswaan).

## List Progress Kegiatan

Fitur ini digunakan untuk memantau dan mengelola daftar progress kegiatan siswa.

1. Login ke website menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih menu **Progress Kegiatan**.
4. Sistem akan menampilkan halaman progress kegiatan.
5. Pada halaman tersebut, admin dapat:
   - Melihat daftar progress kegiatan.
   - Mengekspor data progress kegiatan.

## Detail Progress Kegiatan

Fitur ini digunakan untuk mengelola daftar progress kegiatan siswa.

1. Login ke website menggunakan akun Admin.
2. Buka menu **Kesiswaan**.
3. Pilih menu **Progress Kegiatan**.
4. Sistem akan menampilkan halaman progress kegiatan.
5. Pilih lalu klik pada nama di daftar progress kegiatan.
6. Sistem akan menampilkan **Detail Progress**.
7. Pada halaman tersebut, admin dapat:
   - Melihat daftar riwayat progress kegiatan.
   - Edit riwayat kegiatan.
   - Hapus riwayat kegiatan.
   - Tambah riwayat kegiatan.
   - Melihat grafik progress kegiatan.

## Edit Progress Kegiatan

Fitur ini digunakan untuk mengedit data progress kegiatan.

1. Login menggunakan akun Admin.
2. Masuk ke menu **Kesiswaan**.
3. Pilih submenu **Progress Kegiatan**.
4. Sistem akan menampilkan halaman daftar progress kegiatan.
5. Tentukan data progress kegiatan yang akan diedit.
6. Klik opsi **Edit** pada data yang dipilih.
7. Isi form edit sesuai dengan kebutuhan.
8. Klik tombol **Simpan** untuk menyimpan data yang telah diperbarui.

## Hapus Progress Kegiatan

Fitur ini digunakan untuk menghapus data progress kegiatan siswa dari sistem.

1. Login menggunakan akun Admin.
2. Masuk ke menu **Kesiswaan**.
3. Pilih submenu **Progress Kegiatan**.
4. Sistem akan menampilkan halaman daftar progress kegiatan.
5. Tentukan data progress kegiatan yang akan dihapus.
6. Klik opsi **Hapus** pada data yang dipilih.
7. Sistem akan menampilkan konfirmasi penghapusan data.
8. Klik tombol **Hapus** untuk melanjutkan proses penghapusan.

## Tambah Progress Kegiatan

Fitur ini digunakan untuk menambahkan data progress kegiatan siswa ke dalam sistem.

1. Login menggunakan akun Admin.
2. Pilih menu **Kesiswaan**.
3. Pilih menu **Progress Kegiatan**.
4. Sistem akan menampilkan halaman progress kegiatan.
5. Klik opsi **Tambah Progress Kegiatan**:
   - Isi form sesuai dengan kebutuhan.
   - Klik opsi **Simpan** untuk menyimpan data.
   - Sistem akan otomatis menampilkan progress kegiatan baru pada Daftar Progress Kegiatan.
6. Klik opsi **Import Progress** → isi form sesuai kebutuhan → lalu klik **Simpan**.
7. Klik opsi **Import Riwayat** → isi form sesuai kebutuhan → lalu klik **Simpan**.
