Menu Tugas pada modul Akademik digunakan admin sekolah untuk mengelola tugas siswa, mulai dari daftar tugas, detail dan penilaian, hingga penambahan tugas baru.

## Daftar Tugas

Lihat list daftar tugas yang sudah dibuat.

Admin dapat melihat list daftar tugas, langkah-langkah sebagai berikut :

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. System akan menampilkan daftar tugas.

### Informasi yang ditampilkan

Tabel daftar tugas memuat kolom: **Instansi**, **Tugas** (judul), **Mata Pelajaran**, **Tingkat**, **Kelas**, **Jurusan**, **Tanggal Dibuat**, **Batas Pengumpulan**, **Pengumpulan** (jumlah terkumpul dari total), **Guru**, dan **Status**.

Data dapat disaring dengan filter: **Instansi**, **Tingkat**, **Kelas**, **Jurusan**, **Guru**, **Mata Pelajaran**, rentang **Tanggal Dibuat**, rentang **Tanggal Pengumpulan**, dan **Status** (Ditugaskan / Selesai).

<Callout kind="info" collapsed="false">
  Status tugas ditentukan otomatis dari batas pengumpulan: tugas berstatus **Ditugaskan** selama batas waktu belum lewat, dan **Selesai** setelah batas waktu terlewati.
</Callout>

## Detail Tugas

Lihat detail tugas, siswa yang mengumpulkan, dan berikan penilaian.

Admin dapat melihat detail daftar tugas, langkah-langkah sebagai berikut :

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. System akan menampilkan daftar tugas.
4. Klik tombol **Titik Tiga** di kolom **Aksi**, kemudian klik tombol **Detail**
5. System akan menampilkan detail tugas
6. Pada halaman detail dapat:
   1. Melihat siswa yang mengumpulkan tugas
   2. Memberikan nilai tugas
   3. Katrol nilai tugas

### Informasi yang ditampilkan

Tabel pengumpulan tugas memuat kolom: **Nama Siswa**, **No Kartu**, **Status**, **Waktu Pengumpulan**, **Waktu Penilaian**, **Teks Jawaban**, **File**, **Nilai**, dan **Aksi**. Data dapat disaring berdasarkan **Status** dan diexport ke Excel (file "Daftar Pengumpulan Tugas").

Status pengumpulan yang mungkin:

| Status     | Arti                                            |
| ---------- | ----------------------------------------------- |
| Ditugaskan | Tugas sudah diberikan, siswa belum mengumpulkan |
| Terkumpul  | Siswa sudah mengumpulkan, belum dinilai         |
| Dinilai    | Tugas sudah diberi nilai                        |

<Callout kind="info" collapsed="false">
  Nilai tugas harus berupa angka **0–100**. Saat guru mengumpulkan atas nama siswa: jika tugas berbasis file, lampiran opsional dengan format pdf, doc, docx, xls, xlsx, ppt, pptx, zip, rar, jpg, jpeg, png, heic, heif (maks. 20 MB); jika tidak, jawaban diinput sebagai teks. Saat mengubah penerima tugas, minimal 1 siswa harus dipilih.
</Callout>

## Tambah Tugas

Buat data tugas baru untuk siswa.

Admin dapat menambah data tugas baru, langkah-langkah sebagai berikut :

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. System akan menampilkan daftar tugas.
4. Klik tombol **Tambah Tugas**
5. Maka sistem akan menampilkan form **Tambah Tugas**
6. Isi semua form **Tambah Tugas**
7. Kemudian klik **Simpan**

### Isian formulir

| Field                           | Wajib | Keterangan                                                                                                                              |
| ------------------------------- | ----- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Instansi                        | Ya    | Pilih instansi tempat tugas dibuat                                                                                                      |
| Guru                            | Ya    | Cari berdasarkan nama/nomor kartu guru; jadwal mengajar guru akan dimuat otomatis                                                       |
| Judul Tugas                     | Ya    | Maksimal 255 karakter                                                                                                                   |
| Pilih Kelas Mengajar            | Ya    | Minimal 1 kelas dari jadwal mengajar guru; dapat memilih lebih dari satu                                                                |
| Penerima tugas                  | Ya    | Per kelas: semua siswa (default) atau pilih siswa tertentu (minimal 1 siswa)                                                            |
| Pengumpulan tugas berbasis file | Ya    | Menentukan siswa mengumpulkan berupa file atau teks jawaban                                                                             |
| Batas waktu                     | Ya    | Tanggal & jam; tidak boleh sebelum hari ini                                                                                             |
| Catatan / Deskripsi / Petunjuk  | Ya    | Teks bebas                                                                                                                              |
| Lampiran                        | Tidak | Format: pdf, doc, docx, xls, xlsx, ppt, pptx, zip, rar, jpg, jpeg, png, heic, heif; maksimal 20 MB per file; dapat lebih dari satu file |

<Callout kind="info" collapsed="false">
  Jika lebih dari satu kelas dipilih, sistem membuat satu tugas terpisah untuk setiap kelas. Daftar siswa penerima mengikuti tingkat/kelas/jurusan jadwal; jika jadwal memakai tag, hanya siswa dengan tag yang sama yang muncul.
</Callout>

## Log Device Attendance

Lihat riwayat kehadiran pada absensi wajah.

Fitur ini digunakan untuk pengguna melihat riwayat kehadiran pada absensi wajah dengan cara sebagai berikut:

1. &#x20;Login ke website
2. Kemudian klik Tab **Akademik**
3. Kemudian klik **Log Device Attendance**
4. Kemudian system akan menampilkan form **daftar Log Device Attendance**
5. Kemudian klik **filter&#x20;**&#x75;ntuk meyaring data yang ditampilkan berdasarkan kriteria tertentu.
6. Kemudian klik **export** mengunduh data kehadiran
