---

Menu Tugas pada modul Akademik digunakan admin untuk mengelola tugas siswa — mulai dari daftar tugas, detail dan penilaian, hingga penambahan tugas baru.

## Daftar Tugas

Admin dapat melihat list daftar tugas dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. Sistem akan menampilkan daftar tugas.

## Detail Tugas

Admin dapat melihat detail daftar tugas dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. Sistem akan menampilkan daftar tugas.
4. Klik tombol **Titik Tiga** di kolom **Aksi**, kemudian klik tombol **Detail**.
5. Sistem akan menampilkan detail tugas.
6. Pada halaman detail, admin dapat:
   - Melihat siswa yang mengumpulkan tugas.
   - Memberikan nilai tugas.
   - Katrol nilai tugas.

## Tambah Tugas

Admin dapat menambah data tugas baru dengan langkah-langkah berikut:

1. Masuk ke website, kemudian klik menu **Akademik**.
2. Pilih submenu **Tugas**.
3. Sistem akan menampilkan daftar tugas.
4. Klik tombol **Tambah Tugas**.
5. Sistem akan menampilkan form **Tambah Tugas**.
6. Isi semua form Tambah Tugas.
7. Kemudian klik **Simpan**.

## Log Device Attendance

Fitur ini digunakan pengguna untuk melihat riwayat kehadiran pada absensi wajah, dengan cara berikut:

1. Login ke website.
2. Kemudian klik tab **Akademik**.
3. Kemudian klik **Log Device Attendance**.
4. Sistem akan menampilkan daftar **Log Device Attendance**.
5. Klik **Filter** untuk menyaring data yang ditampilkan berdasarkan kriteria tertentu.
6. Klik **Export** untuk mengunduh data kehadiran.
