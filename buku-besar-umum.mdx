Menu **Buku Besar Umum** digunakan untuk mengelola data buku besar umum yang digunakan pada sistem. Admin dapat melihat daftar buku besar umum, menambahkan buku besar umum baru, serta mengubah, menghapus, menyaring, mencari, mengekspor, dan mengimpor data buku besar umum yang telah dibuat.

<Callout kind="alert" collapsed="false">
  Pastikan Anda sudah login sebagai Admin dan memiliki hak akses ke menu **Pengaturan → Pembukuan → Buku Besar Umum**.
</Callout>

**List Buku Besar Umum**

Halaman **Buku Besar Umum** digunakan untuk melihat daftar seluruh akun buku besar umum yang terdaftar pada sistem beserta informasi posisi normalnya.

link dokumentasi

1. **Buka menu Buku Besar Umum**

Pada sidebar sebelah kiri, klik **Pengaturan → Pembukuan → Buku Besar Umum**.

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784862655787-kidevrul96-list--buku-besar-umum-.png?fm=auto&auto=compress%2Cformat" width="1366" height="682" alt="" />

2. **Lihat daftar Buku Besar Umum**

Sistem akan menampilkan tabel data dengan informasi **Nama Akun**, **Nomor Akun**, **Buku Besar**, **Posisi Normal** (_DEBIT/CREDIT_), dan kolom **Aksi**.

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784863771762-y3c25xzicg-list-data--buku-besar-umum-.png?fm=auto&auto=compress%2Cformat" width="1350" height="684" alt="" />

3. **Cari data Buku Besar Umum**

Gunakan kolom **Cari** di sudut kanan atas tabel untuk menemukan akun secara cepat berdasarkan kata kunci nama atau nomor akun.

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784865147198-l6f4q90d4id-pencarian---buku-besar-umum-.png?fm=auto&auto=compress%2Cformat" width="1350" height="684" alt="" />

4. **Filter data Buku Besar Umum**

Gunakan opsi filter di atas tabel untuk memfilter data:

- **Buku Besar:** Memfilter data berdasarkan kelompok buku besar.
- **Posisi Normal:** Memfilter data berdasarkan posisi akun (**DEBIT** atau **CREDIT**).

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784879012322-cc233dxvn6p-filter-buku-besar-umum.png?fm=auto&auto=compress%2Cformat" width="1348" height="661" alt="" />

**Edit Buku Besar Umum**

Fitur **Edit Buku Besar Umum** digunakan untuk mengubah atau memperbarui data buku besar umum yang sudah ada pada sistem. Admin dapat menyesuaikan kembali informasi akun seperti buku besar, nama akun, nomor akun, maupun posisi normal.

link dokumentasi

1. **Buka menu Buku Besar Umum**

Pada sidebar sebelah kiri, klik **Pengaturan → Pembukuan → Buku Besar Umum**.

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784862655787-kidevrul96-list--buku-besar-umum-.png?fm=auto&auto=compress%2Cformat" width="1366" height="682" alt="" />

2. **Klik ikon Edit pada tabel**

Pada halaman daftar **Buku Besar Umum**, cari akun yang ingin diubah. Pada kolom **Aksi**, klik ikon **Edit** (gambar pensil).

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784879514313-rypdg6hu2vf-edit-buku-besar-umum.png?fm=auto&auto=compress%2Cformat" width="1348" height="661" alt="" />

3. **Perbarui formulir data Buku Besar Umum**

Ubah atau sesuaikan informasi pada formulir yang muncul:

- **Buku Besar:** Pilih kelompok buku besar.
- **Nama Akun:** Ketik nama akun.
- **Nomor Akun:** Ketik nomor/kode akun.
- **Posisi Normal:** Pilih posisi normal (**DEBIT** atau **CREDIT**).

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784879795618-el6jrs17xfq-edit-data-buku-besar-umum.png?fm=auto&auto=compress%2Cformat" width="1366" height="681" alt="" />

4. **Simpan perubahan data**

Setelah data disesuaikan dengan benar, klik tombol **Simpan** untuk memperbarui data ke dalam sistem.

<Image src="https://blob-cdn.documentation.ai/org-9f0069f2-e3f7-48a0-be6a-ee3f46a1d606/doc-0793c7e9-51db-4bf5-85cf-91406e00f125/1784879997915-mrciemsfa5-simpan-buku-besar-umum.png?fm=auto&auto=compress%2Cformat" width="1366" height="681" alt="" />
