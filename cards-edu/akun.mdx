---

Semua pengaturan akun diakses dari **Profil** (ikon di kanan bawah Beranda).

## Ubah PIN Aplikasi

1. Klik **Profil** → **Ubah PIN Aplikasi**
2. Masukkan PIN lama dan PIN baru, klik **Simpan**

## Reset PIN Aplikasi

1. Klik **Profil** → **Reset PIN Aplikasi**
2. Konfirmasi dengan klik **Reset PIN**
3. Pilih metode pengiriman kode: **WhatsApp** atau **Email** yang tertaut

## Edit Akun (Ganti Email / Nomor HP)

1. Klik **Profil** → **Edit Akun**
2. Klik **Ganti Email** atau **Ganti No**
3. Masukkan data baru beserta **foto KTP** untuk verifikasi
4. Klik **Kirim Pengajuan**

## Hapus Akun

1. Klik **Profil** → **Edit Akun**
2. Klik gambar profil merah di pojok kanan atas
3. Isi alasan penghapusan, klik **Hapus**

> Proses penghapusan maksimal 1×24 jam. **Semua data transaksi ikut terhapus.**

## Bahasa & Tema

- **Bahasa**: Profil → **Bahasa** → cari bahasa → **Terapkan**
- **Tema**: Profil → toggle pada bagian **Mode** untuk gelap/terang

## Akun Terhubung (Google/Apple/Dana)

1. Klik **Profil** → **Akun Terhubung**
2. Untuk **Dana**: klik Dana → **Hubungkan** → masukkan nomor telepon → verifikasi kode via WhatsApp

## Bantuan

- **Hubungi CS**: Profil → **Hubungi CS** → mulai percakapan di kolom chat
- **Syarat & Ketentuan** dan **Kebijakan Privasi** juga tersedia di halaman Profil

## Logout

Profil → gulir ke bawah → klik **Keluar**
