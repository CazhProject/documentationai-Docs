Menu Billing di Cazhpos Dashboard dipakai admin partner untuk memperpanjang paket berlangganan yang sedang berjalan atau membeli paket baru dari web back office.

## Kelola Paket

Paket berlangganan menentukan masa aktif akun serta batas penggunaan CazhPOS (jumlah toko, kasir, produk, dan varian). **Perpanjang Paket** menambah masa aktif paket yang sedang berjalan, sedangkan **Beli Paket** mengganti langganan ke paket lain. Gunakan Beli Paket saat kebutuhan usaha melebihi batas paket saat ini.

1. Klik menu **Akun**, lalu klik tab **Billing**.
2. Pilih salah satu:
   - **Perpanjang Paket**: memperpanjang paket berjalan
   - **Beli Paket**: mengganti ke paket baru

<Callout kind="info">
Pembayaran tagihan paket dari sisi aplikasi kasir ada di [Tagihan & Paket di CazhPOS](/cazhpos/tagihan-dan-paket).
</Callout>

<Callout kind="alert">
Saat paket expired, sejumlah fitur terkunci. Lihat [Paket di Glosarium](/referensi/glosarium#paket--billing).
</Callout>
