Cazhpos Dashboard adalah aplikasi **website** (back office) untuk admin partner [CazhPOS](/cazhpos/tentang). Gunakan dashboard ini untuk mengelola produk yang dijual di aplikasi CazhPOS serta melihat laporan dengan mudah.

## Fitur

1. **Toko**: aturan tiap toko, jam operasional, cabang
2. **Produk**: tambah/kurangi produk dan kelola harga
3. **Karyawan**: tambah, hapus, ubah data karyawan
4. **Inventory**: kelola stok tiap produk
5. **Pelanggan**: data pelanggan dan transaksinya
6. **Kartu**: data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards), top-up dan redeem
7. **Laporan**: otomatis dari tiap transaksi

<Callout kind="info">
Pengelolaan operasional harian (kasir, stok opname, pre-order) dilakukan di aplikasi [CazhPOS](/cazhpos/tentang).
</Callout>
