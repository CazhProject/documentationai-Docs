Menu Pelanggan di Cazhpos Dashboard dipakai admin partner untuk mengelola data pelanggan secara massal (import/export) dan meninjau detail pelanggan dari web back office.

## Kelola Pelanggan

Kelola Pelanggan mencakup import data pelanggan secara massal, penghapusan, export, dan peninjauan detail pelanggan beserta riwayat transaksinya. Data pelanggan di sini sama dengan yang dipakai kasir saat memilih pelanggan pada transaksi di [CazhPOS](/cazhpos/pelanggan-dan-hutang). Import berguna saat memindahkan banyak data pelanggan sekaligus ke dalam sistem, sedangkan export mengunduh salinan data untuk diolah di luar sistem.

<Steps>
  <Step title="Login">Login ke website partner.</Step>
  <Step title="Buka menu Pelanggan">Klik menu **Pelanggan**. List pelanggan terdaftar tampil.</Step>
  <Step title="Kelola dari list">Dari list, admin dapat melakukan: **Import pelanggan**, **Hapus pelanggan**, **Export data pelanggan**, dan **Detail data pelanggan**, termasuk riwayat transaksi.</Step>
</Steps>

<Callout kind="alert">
Hati-hati saat menghapus pelanggan: pelanggan yang dihapus hilang dari daftar dan tidak lagi dapat dipilih pada transaksi di kasir.
</Callout>
