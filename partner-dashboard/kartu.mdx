## Lihat Data Kartu

1. Login website partner
2. Klik menu **Kartu** — semua data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards) tampil
3. Dari sini admin dapat melakukan **top-up dan redeem** kartu

## Export Data Kartu

1. Klik menu **Kartu** → **Export Data Kartu**
2. Sistem meng-export semua data kartu ke file Excel

Operasi kartu di kasir (aktivasi, isi, tarik, cek) ada di [CazhPOS — Saldo & Kartu](/cazhpos/saldo-dan-kartu).
