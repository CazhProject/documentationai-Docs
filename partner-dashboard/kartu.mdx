Menu Kartu di Partner Dashboard dipakai admin partner untuk melihat data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards), melakukan top-up dan redeem, serta meng-export data kartu dari web back office.

## Lihat Data Kartu

<Steps>
  <Step title="Login">Login ke website partner.</Step>
  <Step title="Buka menu Kartu">Klik menu **Kartu**. Semua data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards) tampil.</Step>
  <Step title="Top-up dan redeem">Dari sini admin dapat melakukan **top-up dan redeem** kartu.</Step>
</Steps>

## Export Data Kartu

1. Klik menu **Kartu**, lalu klik **Export Data Kartu**.
2. Sistem meng-export semua data kartu ke file Excel.

<Callout kind="info">
Operasi kartu di kasir (aktivasi, isi, tarik, cek) ada di [CazhPOS — Saldo & Kartu](/cazhpos/saldo-dan-kartu).
</Callout>
