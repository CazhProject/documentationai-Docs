Menu Kartu di Cazhpos Dashboard dipakai admin partner untuk melihat data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards), melakukan top-up dan redeem, serta meng-export data kartu dari web back office.

## Lihat Data Kartu

Halaman ini menampilkan seluruh kartu CARDS yang terdaftar pada partner beserta datanya. Dari sini admin dapat langsung mengubah saldo kartu: **top-up** menambah saldo kartu anggota, sedangkan **redeem** menarik (mengurangi) saldo kartu. Keduanya tercatat pada riwayat transaksi kartu, jadi pastikan kartu dan nominal yang dipilih sudah benar.

<Steps>
  <Step title="Login">Login ke website partner.</Step>
  <Step title="Buka menu Kartu">Klik menu **Kartu**. Semua data [kartu CARDS](/referensi/glosarium#kartu-kartu-cards) tampil.</Step>
  <Step title="Top-up dan redeem">Dari sini admin dapat melakukan **top-up dan redeem** kartu.</Step>
</Steps>

## Export Data Kartu

Export mengunduh salinan seluruh data kartu ke file Excel untuk kebutuhan arsip atau pengolahan di luar sistem; data kartu pada sistem tidak berubah.

1. Klik menu **Kartu**, lalu klik **Export Data Kartu**.
2. Sistem meng-export semua data kartu ke file Excel.

<Callout kind="info">
Operasi kartu di kasir (aktivasi, isi, tarik, cek) ada di [Saldo & Kartu di CazhPOS](/cazhpos/saldo-dan-kartu).
</Callout>
